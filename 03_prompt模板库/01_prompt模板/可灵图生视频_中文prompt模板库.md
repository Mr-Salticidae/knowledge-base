# 可灵图生视频 · 中文 prompt 模板库

> 来源:R-07 项目 2026-05-11 全片实跑验证(17 段视频)
> 适用:可灵 Kling 3.0 / Omni(libtv 幕布版)
> 配套:02_参数行为档案 / 可灵Kling3_0_行为规律.md + 04_方法论与洞察 / 图生视频_ForwardOnly原则.md

## 通用规则(所有模板都遵守)

1. **中文 prompt 对位可灵预设词训练**(预设词全中文)
2. **Forward-Only**——只描述从源图状态向后发展
3. **One Movement Per Shot**——一镜一个主导动作
4. **慢动作修饰**——"以极慢的速度 / 缓慢 / 极缓 / 几乎察觉不到"
5. **接受默认微动**——不堆否定词

## 模板 A · 完全静止(凝固镜头)

**适用**:题眼镜头 / 凝固高潮瞬间 / 长时长(15s)/ 视觉重量最大的镜头

**预设词组合**:
- 基础镜头:**固定镜头**
- 运镜速度:**慢速**
- 景别:**中景**(或按需特写 / 远景)
- 光影:**自然光**
- 画面:**丰富细节**
- 氛围:**宁静**

**Prompt 框架**:
```
[主体描述 - 简洁,呼应源图]。镜头在整个 [X] 秒内完全静止、锁定不动。[主体] 在 [X] 秒内完全静止,没有任何身体动作——[列举关键不动部位]。[次主体] [如何静止]。背景中仅有 [极轻微的元素] 缓慢飘动,几乎察觉不到。镜头不发生任何位移——没有推、拉、摇、移、抖动、漂移。油画质感保留,[色调描述]。绝对静止的电影感,时间冻结的瞬间。
```

**实例**:R-07 S8 题眼 15s
```
一个机器人侧面站立在浓雾弥漫的工业废墟中,蝴蝶停在它的红色圆盘面甲附近。镜头在整个 15 秒内完全静止、锁定不动。机器人的身体在 15 秒内完全静止,没有任何身体动作——头部不转动、肩膀不起伏、躯干不晃动、装甲板不震颤、颈部线缆不甩动、不呼吸、不调整重心。蝴蝶翅膀完全合拢且静止,15 秒内没有任何扇动。背景中仅有极轻微的雾气粒子缓慢飘动,几乎察觉不到。镜头不发生任何位移。油画质感保留,冷灰色调与雾蓝色调。绝对静止的电影感,时间冻结的瞬间。
```

## 模板 B · 微动静立(克制日常存在)

**适用**:R-07 静立 / 主体不变但环境有微动 / "活着但不张扬"

**预设词**:同模板 A

**Prompt 框架**:
```
[主体]静静地 [姿态描述] 在 [场景],[环境元素]。镜头在整个 [X] 秒内完全静止、锁定不动。[主体] 的身体在 [X] 秒内完全静止,没有任何身体动作——[关键不动部位]。仅有极轻微的微动:[元素 1] 在 [情景] 中以极小幅度 [运动描述]、[元素 2] [运动描述]、[元素 3] [运动描述]。除此之外没有其他动作。油画质感保留,[色调]。
```

**实例**:R-07 S2 (R-07 静立 8s + 颈缆 + 雾流 + 草微动)
```
一个机器人静静地站立在杂草丛生的废墟空地中央,以背影/侧背的角度入画,远处是半坍塌的工厂厂房,周围有混凝土裂缝中长出的野草、苔藓覆盖的钢梁与藤蔓。镜头在整个 8 秒内完全静止、锁定不动。机器人的身体在 8 秒内完全静止,没有任何身体动作——不行走、头部不转动、肩膀不起伏、腿部不动。仅有极轻微的微动:颈部线缆在轻风中以 1-2 厘米的极小幅度缓慢摆动、背景大气雾气向上缓慢飘动、画面边缘的前景野草在微风中极轻地摇曳。除此之外没有其他动作。油画质感保留,冷灰色调与雾蓝色调,后启示录的静谧感。
```

## 模板 C · 凝固对望(双主体)

**适用**:双主体不动 + 时序分段微动 / "时间被凝固的相遇"

**预设词**:同模板 A

**Prompt 框架**:
```
镜头在整个 [X] 秒内完全静止、锁定不动。[主体 A] 和 [主体 B] 保持完全静止的对望状态——[A 姿态],[B 姿态]。[A] 在 [X] 秒内完全静立——[A 关键不动部位]。[X] 秒内,[B] 的 [具体微动部位] 在第 [X-Y] 秒之间轻微抽动 1 次,[另一部位] 在第 [Z-W] 秒之间轻微抽动 1 次,**总共仅 2 次微动**,除此之外完全静止。油画质感保留。
```

**实例**:R-07 S14 兔子相遇 13s
```
镜头在整个 13 秒内完全静止、锁定不动。机器人和墙顶上的小兔子保持完全静止的对望状态——机器人面向左侧的塌墙,身体不动、头部不动、手臂不动、双腿不动,机械传感器与兔子的眼睛保持在同一高度线上的平视对望。兔子始终在墙顶原位不动,毛色保持原图的灰褐色调,与环境融合,不被美化变浅。13 秒内,兔子的耳朵在第 4-5 秒之间轻微抽动 1 次,鼻子在第 8-9 秒之间轻微抽动 1 次,除此之外完全静止。机器人在整个 13 秒内完全静立——不行走、不转身、不移动位置、不挥动手臂、颈缆几乎不动。冷灰雾蓝色调,后启示录的克制对望,时间被凝固的相遇。
```

> **注意**:这是从"R-07 走入开阔地停下转向兔子对望"(反向逻辑)修订的 forward-only 版本。详见 04_方法论与洞察 / 图生视频_ForwardOnly原则.md。

## 模板 D · 小动物飞行 / 蝶轨迹

**适用**:蝶 / 鸟 / 飞虫的飞行轨迹(进入 / 接近 / 落下 / 飞过)

**预设词**:
- 基础镜头:**固定镜头**
- 运镜速度:**慢速**
- 景别:**中景** 或 **近景**

**Prompt 框架**:
```
[飞行主体] 从画面 [起始位置] 缓慢飞入,飞行轨迹呈 [轻微 S 型 / 慢弧线 / 匀速] 起伏,以匀速的慢速向 [终点方向] 移动。[X] 秒内完成"飞入 → 接近 → [停留 / 落下 / 悬停]"的进程,最终在第 [Y-Z] 秒之间 [终点动作]。[主体保持静止描述]。镜头完全静止。冷灰雾蓝色调,[暖色锚点] 作为唯一暖色。油画质感保留。
```

**约束(必加)**:
- 数量严格(单只 / 2-3 只),不要凭空增加
- 轨迹匀速,不俯冲,不盘旋
- 不飞出画面(除非剧情需要)

## 模板 E · 机械主体行走 + 跟拍

**适用**:R-07 类机械主体行走 / 跟拍镜头 / 镜头跟随平移

**预设词**:
- 基础镜头:**跟随**(关键!跟拍镜头与"固定镜头"形成强差异化)
- 运镜速度:**慢速**
- 景别:**远景** 或 **中景**

**Prompt 框架**:
```
镜头跟随 [主体] 缓慢 [向右/向左] 平移,镜头平移幅度约 [%],[X] 秒内体现明显的跟拍感。[主体] 在画面中持续向前行走,机械式步伐分明,每一步有清晰的脚步落下感,颈部线缆与背部装甲随步伐自然摆动。[主体] 在画面中保持在中央偏 [左/右] 的位置(镜头跟随保持构图)。在第 [X-Y] 秒之间,[主体] 逐步走向画面 [出框方向] 的边缘,即将走出框外。画面 [两侧/前景] 的 [环境元素] 在镜头平移过程中自然滑过画面。冷灰雾蓝色调,[主体暖色锚点] 是画面里唯一的暖色锚点。油画质感保留,[主体] 形态严格保持 humanoid 比例。
```

**实例**:R-07 S13a (9s 跟拍向右 + R-07 走出右侧)
```
镜头跟随机器人缓慢向右平移,镜头平移幅度约 35%,9 秒内体现明显的跟拍感。机器人在画面中持续向前行走,机械式步伐分明,每一步有清晰的脚步落下感,颈部线缆与背部装甲随步伐自然摆动。机器人的背影在画面中保持在中央偏左的位置。在第 8-9 秒之间,机器人逐步走向画面右侧的边缘,即将走出框外。画面左侧的苔藓覆盖钢架、画面右侧的小树等环境元素,在镜头平移过程中自然滑过画面。冷灰雾蓝色调,机器人红装甲是画面里唯一的暖色锚点。油画质感保留,humanoid 比例严格保持。
```

## 模板 F · 镜头拉远 + 主体缩小(Outro 类)

**适用**:Outro 段 / 退场感 / 主体在画面中变小

**预设词**:
- 基础镜头:**拉远**
- 景别:**远景**

**Prompt 框架**:
```
镜头以极慢的速度持续拉远,[X] 秒内拉远幅度约 [%]。[主体] 在画面里继续 [行走 / 静止],[X] 秒内从当前的 [起始尺寸] 高度逐步缩小到 [末尺寸] 高度。背景大气雾气在 [X] 秒内逐渐加厚,远处工厂剪影更模糊。[主体暖色] 是画面里唯一的暖色锚点,这个暖色随着主体变小而面积同步缩减,**不要被美化放大**。冷灰雾蓝色调主导,后启示录的辽远感。油画质感保留。
```

**实例**:R-07 S15 (15s 拉远 + R-07 缩小)
```
镜头以极慢的速度持续拉远,15 秒内拉远幅度约 25-30%。机器人在画面里继续向画面深处行走,机械式步伐稳定,颈缆随步伐轻微摆动,机器人因为镜头拉远和自身向前走的双重作用,在画面里从当前的中远景尺寸(约 50% 高度)逐步缩小到大远景尺寸(约 25-30% 高度)。背景大气雾气在 15 秒内逐渐加厚,远处工厂剪影更模糊。机器人的红色装甲是画面里唯一的暖色锚点,这个暖色随着机器人变小而面积同步缩减,不要被美化放大。冷灰雾蓝色调主导,后启示录的辽远感,humming 进入的退场氛围。油画质感保留。
```

## 模板 G · 横向平移(规避中央背景幻觉)

**适用**:无主体的环境镜头 / 当源图中央有可疑模糊轮廓时

**预设词**:
- 基础镜头:**右摇** 或 **左摇**
- 景别:**中景**

**Prompt 框架**:
```
镜头以极慢的速度向 [方向] 平移,[X] 秒内平移幅度约 [%],几乎没有向前推进或拉远,只有横向移动。画面内的 [环境元素] 在镜头平移过程中自然横向滑过画面,左侧的元素逐渐移出画面,右侧新的元素进入画面。前景的 [细节元素] 在微风中轻摇。**画面中央背景的远处元素和任何模糊的阴影都保持远处的模糊状态,不被放大、不变清晰,不会被识别为任何人形或机器人形状**。画面中完全没有机器人,没有任何人形物体出现。冷灰雾蓝色调,以 [低饱和暖色元素] 为暖色锚点。油画质感保留,纯环境镜头,慢速横向扫视。
```

> **关键约束**:推进镜头会让中央阴影被"清晰化为主体"(详见 02_参数行为档案 § 推进镜头的背景幻觉风险)。**当源图中央背景有可疑模糊轮廓时,改用横向平移**。

## 如何使用

1. 写新镜头 prompt 前,**先判断属于哪类模板**(凝固 / 静立微动 / 对望 / 飞行 / 行走跟拍 / 拉远 / 横向平移)
2. 复制对应模板,填入具体细节
3. 过一遍 Forward-Only 自检(prompt 是否在描述"如何到达源图状态"?)
4. 接受"默认微动"——不堆砌反复否定
