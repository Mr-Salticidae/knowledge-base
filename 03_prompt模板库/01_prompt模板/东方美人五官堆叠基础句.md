## 一句话总结

**一段可复用的五官 + 气质堆叠词块**,无需 personalize、无需参考图,靠纯文字就能稳定锁定"复古东方美人"原型(瓷肌血色 / 杏眼深棕瞳 / matte 唇 / 清冷气质),适合**接单 / 时尚 / 胶片向**的高质感东方女性肖像。

---

## 如何使用

✅ **适用主题**:**写实 / 胶片 / 时尚向**的东方女性肖像
- 接单产出、人像写真、胶片质感大片
- 需要"成熟、清冷、有距离感、勾人"气质的脸

✅ **不适用**
- **当代小红书甜妹 / 清冷少女 / 低龄感**——这套词出来偏 25-30 岁成熟妩媚,做少女会偏老,该走 personalize 路线;清冷少女的换轨词块见 [[清冷少女五官堆叠基础句]],机制见 [[AI甜妹脸vs复古东方美人]]
- 极简插画 / 动画风——那是 [[MUJI极简插画基础句]] / [[OVA怀旧基础句]] 的活,本词块是写实摄影向

⚠️ **配合**
- [[AI甜妹脸vs复古东方美人]]——先判断"要当代审美还是复古胶片质感",再决定堆不堆这套词
- 想卖眼神 + 直视镜头时,见 [[主体不看镜头律]] 的"邀请体裁"反例条款

## 基础句词块

> 直接插在"品质开头"和"镜头 / 动作"之间。各 [占位] 按需替换。

```
[an east asian woman / east asian women],
delicate oval face, soft jawline, slightly raised cheekbones,
porcelain-like warm skin with natural crimson,
almond shaped slightly drooping dark brown pupils shimmering with a moist luster,
distant mountain eyebrows, small and delicate nose bridge,
gentle [light rose matte] lips,
a [cold and ethereal elegant] temperament, a refined atmosphere
```

### 完整骨架（含前后段）

```
Museum level exquisite art portrait photography, [情绪/场景],
[上面的五官词块],
[镜头/动作], [发型], [环境],
[胶片/后期尾巴:35mm film / Kodak Portra 400 / deep shadow gradient / fashionable editing 等]
```

---

## 组件分层（每层都可单独替换）

| 层 | 默认词 | 作用 / 可替换方向 |
|---|---|---|
| 脸型骨相 | `delicate oval face, soft jawline, slightly raised cheekbones` | 锁鹅蛋脸 + 骨相;要圆脸娃娃感则删颧骨词 |
| 肤质 | `porcelain-like warm skin with natural crimson` | 瓷肌带血色,这是"高级感"关键;偏冷白皮则 `cool porcelain skin` |
| 眼（核心勾人层） | `almond shaped slightly drooping dark brown pupils shimmering with a moist luster` | 杏眼 + 略下垂 + 深棕瞳 + **潮湿光泽**=楚楚勾人;要清亮去掉 drooping |
| 眉 / 鼻 | `distant mountain eyebrows, small and delicate nose bridge` | 远山眉 + 秀气鼻梁,东方古典感 |
| 唇 | `gentle light rose matte lips` | matte 哑光更高级;要妩媚换 `deep rose / wine red`,要清纯换 `pale pink glossy` |
| 气质（情绪开关） | `a cold and ethereal elegant temperament` | **整段的情绪开关**:cold=清冷距离感;去掉 cold=温暖亲和;换 `sensual / sultry`=妩媚 |

**眼层 + 气质层是这套词的灵魂**——`moist luster` 决定勾人,`cold ethereal` 决定清冷。其余几层是骨架,缺了气质会平。

---

## 气质变体（换"气质层"一个词，整张脸的味道就变）

| 想要的味道 | 气质层写法 | 配套唇色 |
|---|---|---|
| 清冷 / 高级 / 有距离 | `cold and ethereal elegant temperament` | light rose matte |
| 妩媚 / 成熟 / 性张力 | `sultry and alluring sensual temperament` | deep rose / wine red |
| 温柔 / 亲和 / 邻家 | `gentle warm approachable temperament` | soft pink glossy |
| 慵懒 / 疏离 / 文艺 | `languid detached melancholic temperament` | nude matte |

---

## 必须注意的点

- ❌ **别用在"当代甜妹 / 少女"题材**——见上,会偏老偏复古,该走 personalize。
- ⚠️ **审核边界**:本词块本身安全,但配 `bathing in a bathtub` + 裸肩这类场景已接近 MJ 审核线,再往裸露推会触发审核,接单向不要冒进。
- ⚠️ **打光 vs 调色**:尾巴若叠了 `deep shadow gradient` / 胶片色卡,前面写的 `sunny / golden` 基本不生效——调色词会赢,见复盘方法论 1。

## 实证

### 浴缸沐浴东方美人（2026-06-13，一次抽卡即得）

完整 prompt 与复盘见 [[2026-06-13_浴缸沐浴东方美人_Portra胶片人像复盘]]。无 personalize、无参考图,纯文字一次成型,稳定得到瓷肌血色 + 深棕潮湿瞳 + matte 唇 + 清冷气质的复古东方美人原型。

### 露背回眸东方美人（2026-06-13，同词块换皮）

见 [[2026-06-13_露背回眸东方美人_chiaroscuro胶片人像复盘]]。同一词块,眼层写成 `dewy gaze`、气质层写成 `cool ethereal grace`(本模板默认词的近义变体),同样稳定生效——说明词块里各槽位有近义替换空间:
- 眼层潮湿:`shimmering with a moist luster` ≈ `with dewy gaze`
- 气质层清冷:`cold and ethereal elegant temperament` ≈ `cool ethereal grace`

> 两次来源均为胶片写实向,跨更多场景 / 色卡仍待验证。
