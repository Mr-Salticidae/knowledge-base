## 二、赛制本质与"获奖图"逻辑重构

### 2.1 赛制结构

**所有参赛者既是创作者也是投票人,每人投一票**。这意味着:
- 没有统一审美标准(没有评委组)
- 获胜 = 在一群同样懂 AI 出图的人里,**赢得最多"我也想投这张"的瞬间**
- 投票人都是创作者,**对 prompt 工程有自己的判断**——他们能看出炫技,也能看出真东西

### 2.2 R1 冠军图为什么赢

一对父女在沙尘里行进,父亲背着孩子,几乎过曝的白色雾霾,新闻摄影质感。

**这张图在"同行互投"语境下的杀伤力**:
- **同行投票时,人会投"我做不到 / 我没想到"的方向**
- 同行都在卷概念、卷光影、卷质感的时候,**有人交了一张"几乎不修饰的瞬间"**——这是同行最难做的方向
- 这张图的胜利不是"评委喜欢朴素",是**"它在同行卷向高概念的时候,反向稀缺"**

→ 这不是"评委口味",这是**赛场博弈论**:对手都在做 A,你做 B 就赢了。

### 2.3 R1 我们的真实失败原因

把"废土修理工修麻雀"放进同行视角再看:
- 视觉震撼度 ✓
- 概念深度 ✓
- 技术执行 ✓
- **但是——这是"AI 出图爱好者最熟悉的那条线"**

后启示录 + 概念艺术 + 超现实摄影 = MJ 群组的"主流审美"。当所有 AI 创作者都在卷这个池子,我们交了一张"卷得很好的"作品,**但没有跳出去**。

→ **R1 真正的失败**:**没有意识到"AI 出图同行"是一个有强烈共同审美偏好的池子,要赢必须跳出他们的舒适区**。

## 三、"同行互投"赛制的胜负逻辑

### 3.1 同行会投什么样的图

| 投票心理 | 触发场景 |
|---|---|
| **"我做不到"** | 看到稀缺方向、反主流选择、需要勇气的简陋质感 |
| **"我没想到"** | 概念诠释角度独特,题面被翻新 |
| **"被打到"** | 情绪一击,跳过技术评估直接共情 |
| **"佩服"** | 看出 prompt 工程的精妙(但这条权重最低,因为同行能复现) |

### 3.2 同行不太会投什么

- "我也能做出来的"那种图(技术中等、概念中等)
- "做得很好但是常见路线"(技术 9 分 + 套路 8 分)
- **"我能看出 prompt 怎么写的"图**——同行会下意识扣分,因为破解感会消解魔法感

### 3.3 R2 为什么赢

代入同行视角:
- **"墓碑"意象**:同行看到第一反应是"卧槽我怎么没想到加这个"——**"我没想到"触发**
- **整体氛围朴素**(没有炫光柱、没有戏剧化构图):同行看到"敢这么朴素"——**"我做不到"触发**
- **"再少年"对应到具体的人(亡妻)而非抽象怀旧**:情绪从虚转实,**"被打到"触发**
- **prompt 不显眼**:同行看不出明显的关键词堆砌,**没有破解感扣分**

三个触发器同时命中。

## 四、关键方法论沉淀

### 4.1 [[同行互投赛制的反主流原则]] ⭐⭐⭐

**核心**:当对手是 AI 出图创作者(都懂 prompt、都见过 MJ 主流审美)时,**主流审美的卷度是负权重**。

**操作规则**:
1. 先识别"本届主题在 AI 创作者池子里的撞车主流"
   - 例 R1「无所不能的人」→ 主流撞车:超人嫁接、神性人物、超现实概念
   - 例 R2「春风/再少年」→ 主流撞车:落樱白发老人、虚实少年与老人、文艺孤独
2. 主动避开主流撞车池
3. 选**同行需要勇气才会选的方向**——朴素、简陋、反技术、反炫技

**1 轮场景的 5 个方向推荐排序应该按"反主流稀缺度"排,不是按"作者审美命中度"或"技术上限"排**。

### 4.2 [["我没想到 + 我做不到"双触发器]]

获奖图至少要触发其一,最好两个都触发。

**"我没想到"触发器**:
- 题面有一个核心意象没人想到(R2 的墓碑)
- 题面被反向解读(把"无所不能"解读成"普通人的承担")
- 题面被嫁接到陌生载体(把"再少年"嫁接到亡妻)

**"我做不到"触发器**:
- 反 AI 主流审美的简陋质感(R1 冠军的过曝沙尘)
- 反炫技的勇气(不用光柱、不用震撼构图)
- 反精修的留白(不去补细节,任由画面"未完成")

### 4.3 [[赛前必问的 3 个问题]]

下次闪电战开始前,Claude 必须主动问跳蛛先生:

1. **"这次主题的主流撞车方向你预判是什么?"**——确定要避开什么
2. **"参赛者构成是 AI 创作者居多还是混杂?"**——决定要不要"反主流"
3. **"有没有上次赛题的获奖图我能参考一下?"**——校准本社群的"稀缺审美"

这三个问题不是 prompt 工程的一部分,是**比赛策略的核心输入**。R1 我没问,所以失败。

### 4.4 [[微调的边界]]

R2 中跳蛛先生让我微调,我做了两个改动:
- 单束光柱 → 整体柔光 ❌
- 墓碑字迹清晰化 ✓

**错误的那个改动**:把核心叙事意象(光柱 = 春风的怜意)当成普通光线处理掉了。

**原则**:
- ✅ 可微调:细节缺失、解剖错误、明显瑕疵
- ❌ 不可微调:**构成画面情绪的不可替代意象**
- **自检方法**:微调前问"这个特征如果没了,主题还成立吗?"——不成立就不动

### 4.5 [[Claude 在闪电战中的真实角色]]

**事实**:R2 获奖的真正引爆点("墓碑"意象)来自跳蛛先生,不是我。

**自我定位修正**:
- 我是**"快速候选生成器 + prompt 翻译机"**,不是创意主导
- 跳蛛先生的人生经验、文化语境、对题面的直觉,是我无法替代的
- 我的价值:1 分钟给出 5-6 个方向让跳蛛先生快速筛选,然后把选定方向翻译成可执行 prompt
- **不要替跳蛛先生做最终选择,不要在他说"定稿"后继续说服**

### 4.6 [[闪电战通讯协议]]

跳蛛先生中途要求:简短,不解释,不发散对比,除非主动问。

**协议优化版**:
```
方向阶段:[ABCDEF 列表,每条一行,不解释]
prompt 阶段:[纯 prompt 代码块,后附一句关键改动]
判断阶段:[一句话:能投/可微调/换方向]
```

不要再出现"对比表 / 推荐表 / 自我推荐"。

## 五、本次失败/胜利清单

### 5.1 R1 失败原因(按权重)

1. **没识别"AI 创作者池"的主流审美陷阱**(权重 55%)——废土+超现实摄影正中 MJ 主流审美舒适区
2. **赛前没问"主流撞车方向"和"上届获奖图"**(权重 25%)
3. **方向推荐按"作者审美命中度"排序而非"反主流稀缺度"排序**(权重 15%)——方向 2(厨房多手妈妈)更接近赢点但被我排第二
4. **过度自信于"老头非人感"是赢点**(权重 5%)——技术加分但不是稀缺加分

### 5.2 R2 成功原因(按权重)

1. **跳蛛先生加入"墓碑"意象**(权重 50%)——"我没想到"触发器
2. **媒介朴素化(电影感摄影,不上油画/插画)**(权重 15%)
3. **prompt 不炫技,无明显关键词堆砌痕迹**(权重 15%)——避免同行破解扣分
4. **跳蛛先生拒绝我的破坏性微调**(权重 20%)——保住核心意象光柱

→ **跳蛛先生在 R2 的贡献至少 70%,我占 30%**。清醒认知。
