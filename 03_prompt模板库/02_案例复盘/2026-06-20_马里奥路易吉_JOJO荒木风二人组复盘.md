## 这一版验证了什么

它是 [[高辨识二人组的剪影梗图法_v1]](v2 泛化版)的**第二个案例**,把方法从单一画风(水墨剪影)推到了另一极(JOJO 全彩):

- **选人不变**:矮胖 + 瘦高、一眼能认、自带梗(马里奥 = 国民游戏 + JOJO 立ち梗)。
- **第二眼不变**:背景藏母题(绿水管 + 金币),逻辑同 Ep.1 的隐虎。
- **变的只是画风**:水墨逆光剪影 → 荒木飞吕彦的扭身张力全彩。

→ 锚点(身材反差 + 第二眼 + 有张力的媒介)与画风解耦,**二次验证 ⭐⭐**。

## 两个新增子法(本案独有干货)

### 1. IP 绕障:风格点名,角色不点名
MJ 对商标角色(马里奥/路易吉)有**输出形象拦截**——提示语判定安全,但生成图被判违规(任天堂维权收紧)。绕法:
- **画风可点名**:`Hirohiko Araki` / `JoJo's Bizarre Adventure` 是风格引用,放行。
- **角色用原型描述**:红帽红衫蓝背带 + 矮胖 + 胡子(马里奥)/ 绿帽 + 瘦高(路易吉),不写商标名。
- 辨识由**配色 + 体型 + 道具**扛,不靠名字。

### 2. personalize 是质感临门一脚
同一条 prompt,叠不叠账号 personalize,是"能看"与"作者非常喜欢"的差别。personalize = 给 MJ 账号训练的专属审美底盘,决定成片质感与张力。详见 [[personalize与moodboard分工]]。

## 可迁移结论

- "二人组梗图"的赢点在**选人 + 格式**,不在画风;锁定身材反差 + 第二眼母题后,画风是自由变量。
- 遇 MJ 商标角色拦截,优先"风格点名 + 角色原型描述",而非放弃题材。
- niji 7 的 precision prompting 适合咬住强风格(荒木);最后用自有 personalize 提质感。

> 验证状态:二次验证 ⭐⭐。第三个画风/题材(如黑白无常 浮世绘)再成立则升铁律。
