## 作品回顾

### 创作思路

做一张接单向的高质感东方美人胶片肖像。核心卖点:潮湿、私密、有距离感的勾人眼神,带 Portra 400 的低饱和暖雾质感。

### 执行过程

纯文字 prompt,一次 / 少量抽卡即得。没有用参考图,没有 sref / oref,没有多轮改词——这说明这条 prompt 的结构本身足够稳,各层锚点之间没有强冲突(除了下面要讲的调色与打光那一处)。

### 最终输出

俯拍角度,东亚女性在浴缸中抬眼直视镜头,湿发凌乱低垂,裸肩,瓷感皮肤带潮湿光泽,眼神潮湿且略带忧郁 / 张力。整体低饱和、暖中偏冷灰的胶片雾感,白窗框 + 虚化家居背景。

## 结果分析（我的方案视角）

> 接单产出,无对手方案,只做"我的方案哪里对上、哪里没对上"的自评。

### 对上预期的部分

- **五官堆叠成功锁脸**:杏眼下垂 + 深棕瞳 + moist luster + 远山眉 + light rose matte lips + cold ethereal temperament,这一整段把脸稳定锁成了"复古东方美人"原型,瓷肌带血色的质感到位。
- **潮湿质感超预期**:湿发、湿肤、潮湿眼神三者统一,这是全图最强的资产(详见方法论 2)。
- **俯拍 + 浴缸的私密感成立**:from above + bathing in a bathtub + 裸肩,营造出"瞥见她的私人时刻"的脆弱感,且没有越过审核线。

### 没完全对上的部分（如实记录，用户判断仍属"基本对上"）

- **调色偏离字面打光**:prompt 第 2 段写了 `sunny bathroom` `golden twilight`(明亮暖阳),但成图明显是低饱和、暖中带冷灰的雾感,不是"阳光金色暮光"。原因是第 7 段的 `deep shadow gradient` + `fashionable editing aesthetic` + `Portra 400` 把亮度和饱和度压下去了——调色 / 后期词压过了打光词(详见方法论 1)。这不是 bug,反而是这张图高级感的来源,但要意识到"sunny / golden"在这里基本没生效。
- **情绪比字面更重**:prompt 写 `quiet contemplation`(平静沉思),成图的直视眼神更接近"忧郁 / 有张力 / 勾人",而非纯然平静。这同样是加分项,但和字面有偏差。

## 方法论沉淀

### 1. 调色 / 后期词会压过打光词 ⚠️

**核心**:当 prompt 同时出现"打光词"(sunny / golden twilight / bright)和"调色 / 后期词"(deep shadow gradient / fashionable editing / 某胶片色卡),成图通常服从后者——胶片色卡和后期风格决定最终明度与饱和,字面打光让位。

**来源**:本图 `sunny bathroom, golden twilight` 几乎未生效,成图被 `deep shadow gradient + fashionable editing + Portra 400` 拉成低饱和暖冷灰雾感。

**验证状态**:首次发现 / 待验证 ⚠️

**操作规则**:
1. 想要明亮阳光感,不要在尾巴叠 `deep shadow gradient` / 大量胶片颗粒 / 暗调后期词,它们会赢。
2. 想要胶片雾感 / 高级灰,就大胆写后期词,不用纠结前面的 sunny 会不会冲突——它基本不会赢。
3. 打光和调色想各管一摊时,挑一个主导,另一个只写不和它打架的词(如 `soft window light` + `muted film grade` 是协同,`harsh sunlight` + `deep shadow` 是打架)。

**反例 / 边界**:不确定是否对所有色卡都成立(Portra 本身就是低饱和暖调,可能放大了这个效应);Fuji / 高饱和色卡 + sunny 是否同样被压,需另测。

### 2. 潮湿三件套：湿发 + 湿肤 + 潮湿眼神，统一出勾人质感 ⚠️

**核心**:沐浴 / 雨中 / 出水类人像,把"湿润"做成跨部位统一的质感系统(头发湿、皮肤湿、眼睛湿),比单点湿更勾人;其中 `moist luster pupils`(潮湿光泽的瞳孔)是眼神勾人的关键词。

**来源**:本图 wet hair + porcelain skin + `almond pupils shimmering with a moist luster` 三者统一,潮湿眼神是用户和我都判断"超预期"的部分。

**验证状态**:首次发现 / 待验证 ⚠️

**操作规则**:
1. 眼神想潮湿勾人,写 `pupils shimmering with a moist luster` / `glossy moist eyes`,别只写 `beautiful eyes`。
2. 配套把湿润铺到头发(`wet / damp hair`)和皮肤(`dewy / moist skin`),三件套统一。
3. 杏眼 + 略下垂(`slightly drooping`)+ 深棕瞳,是"楚楚 / 勾人"组合;想要清冷就配 cold temperament,想要妩媚就去掉 cold。

**反例 / 边界**:干燥 / 冷冽 / 大漠 / 雪原类主题不要乱加湿润词,会破题。

### 3. "复古东方美人"可以被纯文字主动调用（不只是 bias 陷阱）⭐⭐

**核心**:[[AI甜妹脸vs复古东方美人]] 把 MJ base model 默认输出的"民国老照片质感东方美人"定义为 personalize 缺位下的偏差陷阱。本图反向印证:当目标受众不是当代小红书甜妹、而是时尚胶片 / 接单产出时,这套"复古东方美人"质感可以靠纯文字五官堆叠 + 胶片尾巴主动、稳定地调用出来——同一个机制,换个场景就从"陷阱"变成"可用资产"。

**来源**:本图无 personalize、无参考图,纯文字一次抽卡即稳定得到瓷肌血色、深棕瞳、matte 唇、cold ethereal 气质的复古东方美人原型。

**验证状态**:二次验证 ⭐⭐(机制与原档一致,场景从"该避"扩展到"可主动用")

**操作规则**:
1. 接单 / 胶片 / 时尚向,要这种气质,就堆:`porcelain skin with natural crimson` + `dark brown pupils` + `light rose matte lips` + `cold and ethereal elegant temperament` + 胶片色卡。
2. 但凡目标是"当代东方甜妹 / 小红书定位",这套词反而是要避开的——参见原档,该开 personalize。
3. 一句判断:先问"我要的是当代审美还是复古胶片质感",再决定堆不堆这套词。

**反例 / 边界**:这套词出来的脸偏 25-30 岁成熟妩媚,做"清冷少女 / 低龄感"会偏老,需另调。

### 4. 浴缸俯拍 + 直视镜头 = "邀请"体裁，是看镜头律的合法反例 ⭐⭐

**核心**:[[主体不看镜头律]] 说私密时刻类主题主体绝不能看镜头,但留了一个反例条款——"邀请观众进入"的肖像直视。本图就是这个反例的干净样本:浴缸私密场景 + 直视镜头,之所以成立,是因为潮湿勾人的眼神本身就是卖点,体裁是"她在和你对视"而非"瞥见她的秘密"。

**来源**:本图主体抬眼直视镜头,私密感没破,反而靠对视强化了勾人卖点。

**验证状态**:二次验证 ⭐⭐(确认原档反例条款,补一个肖像直视的实例)

**操作规则**:
1. 想要勾人 / 对视 / 卖眼神,主动写 `looking at camera` / `direct gaze`,并把眼神质感词拉满(moist luster)。
2. 想要"瞥见她的私人时刻"的偷窥感,反过来删掉直视词,走看镜头律的"私密三件套"。
3. 一张图只能占一头:要么邀请对视,要么偷瞥私密,别混。

**反例 / 边界**:见原档——纯私密 / 日记 / 沉浸创作类主题,直视会破题。

## 推荐保留 prompt 方向

> 保留当前图优点的稳定版本,把"调色压打光"的矛盾点显式化(去掉无效的 sunny/golden,留下真正生效的胶片雾感)。

```text
Museum level exquisite art portrait photography, an east asian woman bathing in a bathtub, photographed from above, looking up at the camera with a moist glossy gaze, delicate oval face, soft jawline, porcelain skin with natural crimson, almond shaped slightly drooping dark brown pupils shimmering with a moist luster, distant mountain eyebrows, light rose matte lips, cold and ethereal elegant temperament, long wet black hair in a soft messy low ponytail, loose strands at the nape, soft drooping middle bangs, dewy moist skin, white window frame, slightly blurred home shelf background, soft diffused window light, 35mm film, delicate warm toned particles, Kodak Portra 400 color palette, deep shadow gradient, muted fashionable editing aesthetic --ar 16:9 --style raw
```

### 关键变量

- 删掉 `sunny bathroom, golden twilight`:它们在原图基本没生效,留着只会让人误判光线方向。
- 改成 `soft diffused window light`:和 `deep shadow gradient` 协同,不打架。
- 显式补 `wet hair` / `dewy moist skin` / `moist glossy gaze`:把"潮湿三件套"写全。
- `looking up at the camera`:承认并强化"邀请对视"的体裁定位。

## 下次同类图的行动清单

1. 先定调色还是定光:想阳光就别叠暗调后期词;想胶片雾感就大胆写,别再写无效的 sunny。
2. 潮湿题材把湿发 / 湿肤 / 潮湿眼神三件套写全,别只写一个 moist eyes。
3. 先想清楚受众是"当代甜妹"还是"复古胶片质感",再决定堆不堆复古东方美人那套五官词。
4. 直视 vs 不看镜头,按"邀请 / 偷瞥"二选一,别两头占。
5. 审核边界:`bathing in a bathtub` + 裸肩这次过审了,但再往裸露推就会撞 MJ 审核线,接单向不要冒进。
