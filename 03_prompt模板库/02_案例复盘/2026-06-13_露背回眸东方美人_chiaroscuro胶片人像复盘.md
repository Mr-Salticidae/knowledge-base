## 作品回顾

### 创作思路

做一张文学气质的东方女性胶片肖像。核心:露背坐姿 + 回眸看向窗外 + 一道戏剧性的明暗光打在肩背上。

### 执行过程

纯文字一次 / 少量抽卡即得,满意。后来随手把这条 prompt 换了几个词(场景、动作、尾巴词),得到了浴缸版——**不是为了改进,是顺手试**。

### 最终输出

俯视略带的背身坐姿,东亚女性坐在木质窗台上,背对镜头、头转向窗外回眸,露背橄榄铜色缎面裙,一道明暗对比(chiaroscuro)斜扫过肩背,暖肤与冷缎面形成色温对比。白窗框 + 虚化家居背景,Portra 400 暖调胶片颗粒,暗角压边,整体保留了戏剧性的光影对比。

## 结果分析（我的方案视角）

> 临时作品,无对手方案。重点放在与浴缸衍生版的 A/B 对照。

### 这张图的赢点

- **肩背那道 chiaroscuro 是 hero**:一道斜光扫过肩背,把背部体积雕出来,是全图最有记忆点的元素。
- **回眸 + 看向窗外 = 偷瞥体裁成立**:她不看镜头、头转向窗外,观众是"瞥见她一个安静时刻"的旁观者,私密感成立。
- **冷暖材质对比**:`cool satin sheen contrasting warm skin`——冷调缎面贴着暖肤,材质 + 色温双重对比,高级感的来源之一。
- **露背缎面比裸露更稳**:这是穿着礼服的露背(fashion editorial 体裁),审核比浴缸版的裸肩更安全。

## 核心:父子两版 A/B 对照（最有价值的部分）

两版同骨架、只换了若干词,成图调子却分叉明显。**关键在于:用户以为是随手换近义词,实际换掉的是决定调子的结构词。**

| 维度 | 父本(本图) | 浴缸衍生版 | 换词造成的结果 |
|---|---|---|---|
| 光影词 | `dramatic chiaroscuro across shoulders and upper back` | 删掉,只剩 `sunny / golden twilight` | 父本强光影**生效**,子版氛围词**被胶片尾巴压平** |
| 尾巴体裁词 | `fashion editorial aesthetic`(杂志摄影) | `fashionable editing aesthetic`(后期 / 滤镜) | 子版偏向滤镜平涂感,父本保留摄影戏剧性 |
| 暗部词 | `deep shadow vignette`(暗角,保中心对比) | `deep shadow gradient`(渐变,压平) | 父本保留对比,子版整体压平去对比 |
| 视线 / 体裁 | 头转向窗外、背身、不看镜头(偷瞥) | 俯拍、抬眼直视镜头(邀请) | 两版正好是 [[主体不看镜头律]] 的两面 |
| 着装 / 审核 | 露背缎面礼服(穿着) | 浴缸裸肩(裸露) | 父本审核更安全 |

**结论**:浴缸版调子变平、戏剧光丢失,不是"风格选择",是随手换词的**无意副作用**。`editorial→editing`、`vignette→gradient`、删掉 `chiaroscuro` 这三处看似近义,实际把一张戏剧光人像换成了平涂滤镜人像。

## 方法论沉淀

### 1. 光影要用"关系词"，别用"天气 / 时段词" ⭐⭐

**核心**:想要画面有明确光影,要写**光影关系词**(chiaroscuro / backlight / rim light / directional light across X);**天气 / 时段词**(sunny / golden twilight / late afternoon)在胶片色卡 + 后期尾巴下基本会被压平,指望不上。

**来源**:父本 `dramatic chiaroscuro across shoulders` 强光影生效;浴缸版只剩 `sunny / golden twilight`,被 Portra + 后期压成平光。两版 A/B 直接对照。

**验证状态**:二次验证 ⭐⭐(父子两版 A/B;同时是浴缸复盘方法论 1「调色压打光」的细化——把"打光词会输"细化成"输的是氛围词,赢的是关系词")

**操作规则**:
1. 要光影效果,写关系词 + 落点:`dramatic chiaroscuro across the back` / `hard window light slicing across her face`。
2. `sunny` / `golden hour` / `late afternoon` 只能当氛围参考,不要当成会生效的打光指令。
3. 想保留对比,暗部用 `vignette`(暗角)而不是 `gradient`(渐变);gradient 会压平。

### 2. 看似近义的尾巴词不是装饰，换词=换调子 ⭐⭐

**核心**:prompt 尾巴的体裁 / 后期词,看着像近义词,实际各管不同的成像逻辑。换词前要知道每个词管什么,否则会无意中换掉整张图的调子。

**来源**:`fashion editorial aesthetic`(杂志摄影戏剧感)↔ `fashionable editing aesthetic`(后期滤镜平涂感),一词之差,子版气质明显变滤镜化。

**验证状态**:二次验证 ⭐⭐(A/B)

**操作规则**:
1. `editorial` = 杂志 / 时尚摄影体裁(保留摄影质感和戏剧性);`editing` = 后期 / 调色 / 滤镜(偏平涂)。
2. `vignette` = 暗角(保中心对比);`gradient` = 渐变(压平)。
3. 衍生新版时,逐词对比尾巴,知道自己换掉了什么再换;别把"换近义词"当成无害操作。

### 3. 冷暖材质对比塑造高级感 ⚠️

**核心**:写服装 / 材质时给出**色温对比**(冷调材质贴暖肤,或反之),比单写材质名更出高级感。

**来源**:`cool satin sheen contrasting warm skin`——冷缎面 + 暖肤的对比是本图质感来源之一。

**验证状态**:首次发现 / 待验证 ⚠️

**操作规则**:
1. 写材质时带色温:`cool satin sheen contrasting warm skin` / `warm wool against cool pale skin`。
2. 缎面 / 丝绸 / 金属这类有反光的材质,加 `sheen` / `subtle highlight` 强化质感。

**反例 / 边界**:全暖调或全冷调的统一氛围图,不要硬塞对比词。

### 4. chiaroscuro 可用于塑形主体体积（不只切空间）⚠️

**核心**:[[光暗作为空间叙事工具]] 讲的是用一道光切"内 / 外"两个世界;本图是另一种用法——用斜扫的明暗把**主体的体积**雕出来(肩背的光带),光在这里是塑形工具,不是空间边界。

**来源**:本图肩背那道 chiaroscuro 把背部体积和姿态读出来,是 hero 元素。

**验证状态**:首次发现 / 待验证 ⚠️

**操作规则**:
1. 想塑形,写 `dramatic chiaroscuro across [身体部位]` + 强方向光来源。
2. 落点要具体(across shoulders and upper back),不要只写 `chiaroscuro lighting` 泛泛。

**反例 / 边界**:需要五官 / 细节全亮的肖像别用强 chiaroscuro,会丢细节(同 [[光暗作为空间叙事工具]] 反例)。

## 下次同类图的行动清单

1. 要戏剧光就写关系词(chiaroscuro / backlight)+ 具体落点,别靠 sunny / golden。
2. 衍生新版时逐词对比尾巴:editorial≠editing,vignette≠gradient,知道换了什么再换。
3. 材质带色温对比词。
4. 视线按"偷瞥(不看镜头 / 转向窗外)"还是"邀请(直视)"二选一,见 [[主体不看镜头律]]。
5. 审核:露背缎面礼服比裸肩 / 浴缸安全,接单向优先用穿着的露背方案。
