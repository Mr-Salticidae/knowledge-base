## 迭代过程

### 第一阶段:从"嗦粉"避开普通美食图

初始题面"嗦粉"如果直接写深夜路边摊、热气、红油、筷子,很容易撞车。第一次有效转向是把"嗦粉"从动作摄影改成视觉隐喻:

- 粉条不是食物特写,而是山水中的瀑布、云、山路
- 红油汤不是普通汤底,而是画面底部的朱砂色水域
- 筷子不是餐具,而是把一座山水从碗里提起来的外力

这个转向让主题从"吃一碗粉"升级为"一碗粉里生成一个东方山水世界"。

### 第二阶段:镜头推进解决张力不足

第一张可用图已经有"粉条山水"概念,但镜头偏中远景,像完整作品展示。后续迭代把视角改成:

- `extreme macro close-up`
- `from inside the bowl`
- `low-angle upward view`
- `camera almost touching the chili oil broth`

这一步让观众不再站在桌外看碗,而是进入碗内,从红油汤面仰视粉山。张力来自尺度错位:日常食物被低机位拍成山岳。

### 第三阶段:用审美秩序压住"奇观堆叠"

近景版本方向正确,但容易出现三个问题:

- 粉条变成湿黏肉感
- 亭子像后贴的小模型
- 红油前景变脏,整体显得混乱

最终 prompt 用一组审美锚点修正:

- `porcelain-white noodle texture`:把粉条从肉感拉回瓷白、干净、可食用
- `Song dynasty shanshui`:让亭台和山水进入更克制的东方美学秩序
- `restrained cinnabar accent`:把红油控制成朱砂点睛,不是满屏脏红
- `jade green-gray background`:用青灰背景提供宋画式留白
- `museum-quality art direction`:把奇观从"有趣"推向"可展陈"

## 当前图的赢点

### 1. 视角让食物获得山岳尺度

最终图最关键的不是"粉条像山",而是机位真的在碗里。前景红油虚化、粉山从低处拔起、筷子从右侧插入,共同制造了仰视感。

如果镜头退回桌面高度,同样的元素会变成食物微缩模型;只有碗内低机位能让粉条获得山岳尺度。

### 2. 画面保留了足够留白

这张图没有把亭台、瀑布、山峰、红油、蒸汽塞满。上半部大面积青绿色背景让粉山有呼吸空间,也让观众的视线集中在主峰和筷子方向。

对这类超现实食物图来说,留白不是空,而是给尺度幻觉留下成立空间。

### 3. 红油被降级为前景色锚

红油没有抢主角,只在底部形成模糊的朱红色带。这个处理保留了"嗦粉"题眼,同时避免画面落入油腻、脏、廉价美食图。

红油的正确角色是色彩锚点,不是细节主体。

### 4. 亭台小到刚好可读

亭台没有变成大建筑群,只以几个小黑影嵌在粉山上。它们让观众读出"山水",但不抢走粉条主体。

这说明微缩符号要小而准:只要观众识别到亭台,概念就成立;放大反而会变成模型景观。

### 5. 负面词有效压住了失败方向

这张图的干净感很大程度来自负面词:

- `distant view, tabletop view, flat lay, normal food photo`:防止回到普通食物摄影
- `clutter, dirty broth, chaotic noodles`:防止红油和粉条乱成一团
- `plastic texture, horror, slimy flesh`:防止粉条产生廉价模型感或肉感

本案例里负面词不是补充,而是保护审美边界。

## 可迁移方法论

### 碗内低机位可以把日常食物变成世界

**核心**:当食物主题容易普通时,把镜头放进容器内部,用低机位和微距尺度把食物改造成可进入的世界。

**来源**:本次从普通"嗦粉"题面转向 `worm's-eye macro view from inside the bowl`,最终让粉条山水获得山岳尺度。

**验证状态**:首次发现 / 待验证

**操作规则**:
1. 先锁机位:`from inside the bowl / camera almost touching the broth / low-angle upward view`。
2. 再锁尺度:`towering vertical scale / immersive perspective / macro close-up`。
3. 主体必须从容器里"升起"或"压向镜头",否则碗内视角不成立。
4. 用负面词排除 `tabletop view / flat lay / normal food photo`。

**反例/边界**:
- 如果目标是菜单、美食广告或真实食欲,过强碗内视角可能不够商业。
- 如果食物本身没有可形成地貌的形态,需要先找能被放大的结构。

### 超现实食物图需要"可食用质感"约束

**核心**:食物超现实化后容易变成肉感、塑料感或模型感;必须显式约束干净、可食用、触感明确的材质。

**来源**:前一轮近景图出现粉条湿黏肉感和微缩模型感,最终用 `porcelain-white noodle texture`、`clean tactile noodle texture`、`not grotesque` 修正。

**验证状态**:首次发现 / 待验证

**操作规则**:
1. 对白色食物可用 `porcelain-white`、`clean tactile`、`glossy but clean`。
2. 避免只写 `realistic texture`,它可能放大湿黏感。
3. 用负面词排除 `slimy flesh / plastic texture / horror`。
4. 食物主体要保留可食用识别,不要完全变成雕塑或怪物。

**反例/边界**:
- 如果目标是恐怖、怪诞或实验艺术,肉感可能反而是资产。
- 如果做纯插画,材质约束可以弱一些,用线条和形状控制即可。

### 色彩锚点要服从审美秩序

**核心**:强颜色不一定要铺满画面;在东方克制审美里,红油这类高饱和元素更适合作为底部朱砂点睛。

**来源**:最终图将红油控制为前景虚化色带,配合青灰背景和瓷白粉山,形成"白山 / 青雾 / 朱砂"的三色秩序。

**验证状态**:首次发现 / 待验证

**操作规则**:
1. 主体色用低饱和或高明度建立秩序,如瓷白、青灰。
2. 高饱和色只放在边缘、前景或局部反光。
3. 对强色使用 `restrained accent`、`softly out of focus`、`at the bottom edge`。
4. 如果强色开始抢主角,用负面词排除 `dirty broth / oversaturated / chaotic`。

**反例/边界**:
- 如果目标是信息流封面或爆款美食图,红油可以更大、更亮。
- 如果主题本身是辣味冲击,克制红色可能削弱题眼。

### 微缩符号只需刚好可读

**核心**:亭台、人物、建筑等微缩符号的任务是提示语义,不是承担主体;越小越容易保持高级感。

**来源**:本图亭台只作为山水识别锚点出现,没有变成大型建筑模型,因此粉条仍是主角。

**验证状态**:首次发现 / 待验证

**操作规则**:
1. 用 `tiny pavilions naturally embedded` 而不是 `large temples`。
2. 微缩符号放在主体转折处,如山脊、瀑布边、雾中。
3. 不要让微缩符号超过主体视觉权重。
4. 如果符号显得后贴,加强 `naturally embedded / carved into / partially hidden by mist`。

**反例/边界**:
- 如果作品主题是微缩城市或玩具世界,建筑可以成为主体。
- 如果观众不熟悉符号,过小可能导致语义不可读。

## 下次同类图的行动清单

1. 先判断题面是否会落入普通摄影;如果会,先找"可被放大的物理结构"。
2. 对容器类食物,优先测试 `inside the bowl / low-angle / macro` 机位。
3. 主题符号不要放大成主体;只保留刚好可读的文化锚点。
4. 用三色秩序管理画面:主体色、背景色、点睛色。
5. 出图后先看三件事:机位是否还在碗内;食物是否仍可食用;红油是否只做点睛。
