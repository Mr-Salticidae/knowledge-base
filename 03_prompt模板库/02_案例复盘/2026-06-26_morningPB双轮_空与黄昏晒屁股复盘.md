## R2 冠军 · 为什么赢（事实层可归因）

1. **雅裹俗成立**——名画雕塑的崇高臀部干「晒屁股」，俗内核翻译成艺术史外壳，会心而不低俗。→ 活合集第 36 条「雅裹俗」（反唯美 / 亮裹险家族新成员）。
2. **MJ 偏离即升级（再验证）**——`buttocks glowing` 被渲染成**内透光灯笼臀**、站立改成**弯腰前倾**、外加一张**回头脸**，三处偏离全加分。
3. **圆形发光主体 + 对比度强制**——暖光臀是暮色冷蓝里唯一发光体，0.5 秒锁焦。
4. **递远排列**——臀灯往纵深递弱，二次发现节奏 + 系列构图复用感。

⚠️ **诚实归因**：获胜的视觉核心（内透光臀灯）是 **MJ 抽卡涌现的，不是 prompt 写出来的**。我的贡献是「选对雅裹俗路线 + 暮色逆光基底」，临门一脚归 MJ。

**附带边界**（活合集第 37 条）：把这个抽卡灵气写回 prompt 想复现，结果**变差**——「偏离即升级 ≠ 偏离可固化」。

## R1 零票 · 为什么输（事实闭合后定论）

两张冠军共性：**单物件直给、零理解门槛、0.5 秒秒懂**（笼空→鸟飞 / 钱包空→穷，各 1 步）。我方真空天空理解链 3–4 步（袋装云→真空→标价→商品化），高阈值，观众划过未读。

**根因不是阈值技术，是评估者错位**（创作者原话）：
> 我们错误地把「AI 的零延时解析」和「创作者的已知意图」当成了观众的能力。观众两样都没有。

→ 立 [[闪电战冷读者校准律_v1]]（⭐⭐ 元方法论），含子条「画面阈值 ≠ 概念阈值」。
→ 活合集第 27 条「双阈值策略」+1 **反向验证**（首个高阈值零票反例）。

**关键厘清**：美学与可读**正交**。R1 = 美学上限拉满（moodboard 加成）+ 可读下限缺失 → 零票。**可读性 > 美学**：美学是加分，可读才是入场票。

## 意外之喜 · moodboard 跨题材审美迁移（干净 A/B）

R1 意外忘关「中国美女写真 moodboard」→ 出图惊艳；关闭后同 prompt → 粗糙。唯一变量 moodboard 开/关。

→ 边界：内容泄漏 → NSFW ban（R2 雕塑被 ban 实证）。建议养**纯质感无人物 moodboard** 规避。
