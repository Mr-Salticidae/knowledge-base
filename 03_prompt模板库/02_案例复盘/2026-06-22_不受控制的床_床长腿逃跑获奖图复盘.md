## 题面破局：「不受控制的床」怎么翻译

主流解读"床乱蹦 / 床飞起来" = 撞车池 + 卡通风险。床是全世界最该"静止、安全、服从"的符号，**"失控"越克制越可信**：让它做绝不该做的事（自己跑），但放进最普通的真实场景。

发散 6 方向后选 ③ 床长腿逃跑（其余：①床当烈马 ②床单成海啸 ④床不服重力 ⑤床成地貌 ⑥床是漩涡）。选它的理由——**失控 = 一个奔逃关系**（床在跑、人在追），而非一堆乱象；走 [[超现实的日常化原则]] 这条同行互投最稳的获奖家族。

## 本案核心：v1 > v2，"读不太清"反而对

出图后我自己先觉得「腿不够明显，第一时间 get 不到」，于是做了 v2 强化腿部清晰度（明确奔马步态、腿离地完整暴露、锐化腿部、模糊只留被单、被单不遮后腿）。

**结果：v1 获奖；v2 我判断反而不如 v1。**

> ⚠️ **事实先行声明**：v2 **从未提交投票**。"v1 > v2"是「v1 实际拿第一」+「创作者审美判断」两条证据的合并，不是头对头票选结果。因此本结论是 ⚠️ **首次发现级**，仅作待观察假设，不得指导下一次决策。

初步解释——纪实超现实 / 氛围路线里，"读不太清"是**氛围资产**：v1 的动态模糊 + 半遮腿 + 夜色，制造"我好像瞥见一张床在跑、可又说不清"的真实抓拍感；v2 把超现实主体交代得太清楚，反而把"深夜偶然抓拍"降级成"插画演示"。

这给闪电战「秒懂题眼」原则划了条边界：
- **视觉欺骗 / 反讽题** → 秒懂题眼，清晰是友军。
- **纪实超现实 / 氛围题** → "秒被氛围击中 + 留一个'那是什么？'的钩子"，过度交代杀死瞥见感。

→ 独立成档 [[氛围题的读不清优先律_v1]]

## 图本身的赢点

1. **失控 = 关系**：奔跑的床 + 狂追的人，一个动作讲完题面，不堆细节。
2. **异常日常化**：荒诞主体放进最普通深夜居民街，纪实质感让它"像真发生过"。→ [[超现实的日常化原则]]
3. **氛围读不清是友军**：动态模糊 + 单钠灯 + 拖曳尾迹，瞥见感 > 说明感。
4. **冷暖 + 长影**：暖钠灯 vs 冷夜色 + 拉长影子，补足速度与不安。
5. **负面词排雷**：`--no cartoon, cute, disney` 提前摁死卡通飞床撞车池。
