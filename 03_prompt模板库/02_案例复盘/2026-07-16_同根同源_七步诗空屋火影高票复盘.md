**创作路径（三步）**：

1. **第一轮发散（字面系）全否**：异树同根 / 天地同构（闪电镜像根系）/ 血脉即根 / 一炉万灯——创作者反馈"都不太好，想法发散一点"。
2. **第二轮发散（跳出字面）**：敌人同根（拳击手影子）/ 棋盘之下 / 年轮之城 / 烟花树 / 面团暴雨 / 一树双琴。
3. **题眼是创作者接的**：由「敌人同根」方向，创作者本人联想到**"本是同根生，相煎何太急"**。随后融合定案：诗的道具系统（豆 / 萁 / 釜 / 火）承载"相煎"，影子承载"同根"，空屋承载"人已离场"。

**MJ 偏离记录（prompt vs 出图）**：

- 写 `two little brothers`（两个小孩）→ 出图为**一大一小**两个身影——被观众直接读成"兄与弟"（兄长身份自带曹丕/曹植的位差）；

## 判断层（事实闭合后）

### 为什么高票：典故 = 预装解码器

概念理解链不短：空屋火煮豆 → 认出豆萁 → 接通《七步诗》→ 影子=兄弟 → "同根"（3–4 步）。按 [[闪电战冷读者校准律_v1]] 的纯阈值判断，这是高危图。但它高票，因为两件事托底：

1. **画面低阈值先收"氛围票"**：暖火 / 空屋 / 牵手影子，0.5 秒就有情绪信号（温暖里带一点寂寞），不懂典故也愿意投；
2. **概念钥匙是全民预装的**：《七步诗》是共享文化记忆，理解链虽长，但每一步都不需要"教"，只需要"触发"——一旦有人点破（lucia），解码在群里**社交扩散**，后续观看者带着谜底进场，高阈值瞬间清零。

### 为什么未夺冠：解码有延迟，冠军属于零延迟

"没看懂 → 被点破 → 赞叹"的时间差就是**含蓄税**。闪电战投票高峰在前，解码扩散在后；先划过的票已经投给了零延迟的图。冷读者律的预测方向**命中**（高阈值 → 难冠军），但本案补充了它没覆盖的变量：**高阈值 × 全民典故钥匙 × 有人当场解码 = 高票**，而不是零票。

### 诚实归因

- 题眼（接到七步诗）是**创作者本人的联想**，AI 的贡献是：两轮发散把路走到「敌人同根 / 影子叙事」跳板上 + 融合时的结构设计（空屋 / 无人 / 影子锁定措辞）+ prompt 工程；
- 出图最动人的两处（一大一小、蒸汽蚀影）是 **MJ 抽卡涌现**——第 25 条「偏离即升级」第四正例，继续"识别 → 保留 → 不固化"姿势；
- 赛前**没有跑「冷读者 0.5 秒协议」**——不是协议失效，是根本没执行。若跑了，会提前知道这是高阈值图，可以配一张直给版做双保险。

## 沉淀

- **新方法论（活合集第 39 条）**：典故题的「出处现场重建」破局法——题面是成语 / 诗句时，不画成语的字面（同根→树根），去画**典故的出处现场**（七步诗→灶间煮豆），观众脑中预装的典故文本会替你完成解码。道具即识别码（釜 / 豆 / 萁），缺席 + 影子放大余味。边界：自带含蓄税，闪电战冠军线慎用、展示线是主场。
- **[[闪电战冷读者校准律_v1]] 首次实战回验**（v1.1）：方向命中 + 新增"解码钥匙 / 社交解码"变量。
- **第 25 条「MJ 偏离即升级」+1 → 第四正例**（一大一小身影 / 蒸汽蚀影）。
- **v8.2 `--preview` 首战记录**：土墙颗粒、火光层次、蒸汽体积感均立住，写实底座是"影子叙事"可信度的前提（影子戏要靠硬光和真实材质撑）。
