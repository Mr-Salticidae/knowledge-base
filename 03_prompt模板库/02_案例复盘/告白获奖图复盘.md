# 笔记 · 「告白」获奖图复盘

## 图本身的赢点

### 1. 构图的"压"
身体从右上斜切到左下,占据画面绝大部分,**没有给观众"喘气"的空间**。这种压迫感和"告白前夜的疲惫"主题高度同构——情绪是从画面外溢出来的,不是通过表情演出来的。

### 2. 动作的"塌"
不是"趴下睡着",是**整个上半身像断了一样塌在桌上**。手臂松垮、肩膀无力、头埋进臂弯——身体语言传递的是"放弃",不是"困"。这一层比写实摄影版本到位,因为动画语言可以稍微夸张身体姿态而不显得做作。

### 3. 一只睁着的眼睛
最锋利的细节。头垂下来但**眼睛是睁着的,茫然看着信纸**——
- 不是睡着 = 不是逃避
- 是清醒着放弃 = 比睡着更沉
- 半遮在头发后面 = 不直白,要观众"发现"

这个细节是这张图情绪的核心引爆点。

### 4. 色调的"褪"
黄绿主调 + 暗蓝阴影,像旧画面褪色,**自带"已经过去很久"的伤感时态**。冷暖对比里没有暖色救场——台灯的暖光只是个孤岛,被冷蓝冷绿包围。

### 5. 纸团的"克制"
只在桌面堆成一线,**没有泛滥到地上和全屏**——之前写实版本翻车的就是纸团失控。这版纸团数量刚好:够说明"写了很多",但不抢戏。

### 6. 笔的"掉"
笔垂在手指间,像下一秒就要落地——**进行时态的崩溃,比静止的"已经睡着"更动**。

### 7. 复古胶片质感叠加
划痕、颗粒、轻微褪色——动画 + 老胶片的双重 vintage,**绕开了新海诚那种过于干净的当代日漫感**,反而更像 80 年代末 OVA 的质感,不撞车。

## prompt 的赢点

### 1. 风格定位精准且不撞车
`1990s anime cel animation style` + `late 80s to early 90s OVA` 这个组合很关键:
- 没用"anime"这种宽泛词(会出当代日漫)
- 没用"Studio Ghibli"(撞车率最高)
- **OVA 这个词锁定了"小众怀旧 + 手绘感"**,审美调性立刻跳出主流

### 2. 工艺词堆得够厚
`hand-drawn cel shading with thick line art` + `visible film grain and scratches` ——把"老动画的物理痕迹"拆成具体可执行的视觉特征,MJ 才能照着造。光说"vintage anime"会出空壳风格。

### 3. 服装锁死了"日常"
`oversized t-shirt and shorts` —— 这是上一版翻车修出来的关键词。**衣着是"在自己房间崩溃"和"棚拍大片"的分水岭**,这个词一定要钉死。

### 4. 颜色方向有锚点
`muted yellow-green nostalgic color palette` —— 不是抽象的"复古",而是直接给颜色坐标。出图色调一致性强。

### 5. 留白战略到位
**没有写表情、没有写眼神、没有写情绪动作**——只写了 `head down on her arms` + `slumped`。剩下的全交给"defeated mood"和风格本身去调味。

这就是上次翻车反过来给的教训:**写实摄影里"defeated atmosphere"会被理解成戏剧化布景,但动画语境里反而会被理解成动作语言本身**。

### 6. 道具密度刚好
desk + paper balls + letter + pen + lamp + window —— 6 个道具,**说够故事但不堆砌**。

## 可固化的方法论

### A. 写实翻车 → 切换媒介可能比改 prompt 更快
当一个主题用写实跑出来反复"过头",不一定是 prompt 没调好,**有时候是主题本身和写实媒介不匹配**。"告白前夜的崩溃"这种内心戏很重的主题,动画的语法天生比写实摄影合适。

下次遇到反复跑不出来的情绪类主题,优先考虑换媒介,不要在写实里死磕。

### B. 动画/插画风格的关键词分层 ✓ 二次验证(2026-05-12「星河如丝般顺滑」)
要稳定出"特定时代/流派"的动画或插画感,prompt 里至少要有:

1. **时代锚** (`1990s`, `late 80s`, `contemporary japanese editorial`)
2. **媒介锚** (`cel animation`, `hand-drawn`, `OVA`, `muji aesthetic`)
3. **工艺锚** (`thick line art`, `cel shading`, `film grain` / `fine clean ink lines`, `flat muted color fills`, `fine textured paper feel`)
4. **色调锚** (具体色系名)

只写一层(比如只说 "anime" 或 "illustration")出来一定是当代主流风。

**已从"建议"升级为"标准流程"**——「星河」MUJI 插画四层全上,媒介纯度同样保住。

### C. 非摄影媒介里"现代化质感"的词要删 ✓ 二次验证(2026-05-12「星河如丝般顺滑」)
比如这次没有写 `cinematic`、没有写 `fashion editorial`、没有写 `detailed rendering`——这些词会把动画"现代化",拉回新海诚 / 京アニ 的当代风,损失复古独特性。

**写实摄影爱用的词,在动画/插画风格里大概率要删一遍**。

「星河」案例进一步扩展:**表情词也属于摄影逻辑产物**,在极简插画里要主动删干净——详见 [[极简插画删表情律]]。

### D. 可复用的"OVA 怀旧"基础句 → 已独立成档

完整模板 + 四层关键词 + 色调词库 + 必删词清单见 [[OVA怀旧基础句]]。

简版骨架(三次验证 ✓✓✓):

```
1990s anime cel animation style, [主体], hand-drawn cel shading with thick line art, visible film grain and scratches, vintage anime aesthetic reminiscent of late 80s to early 90s OVA, [色调锚], [情绪锚]
```

> 姐妹模板:[[MUJI极简插画基础句]](日系极简插画版本,适用静默/意境/抽象触觉类主题)

### E. 主题难度高 → 优先考虑"非常规媒介"破局 ✓✓✓ 三次验证(2026-05-12「星河如丝般顺滑」· 2026-05-14「秘密基地」)
"告白"这种撞车池主题,**媒介选择本身就是差异化**。当所有人都在拍写实电影感,你做 90 年代 OVA 风,光这一层就赢一半。下次遇到红海主题,先想"用什么媒介",再想"用什么概念"。

「星河」案例验证:"少女 + 银河 + 头发"在写实摄影里是 instagram 红海,切到日系 MUJI 极简插画后直接跳出红海池。**这一层的差异比概念差异更省力**。

「秘密基地」案例第三次验证:小红书池里"儿童秘密基地"是小孩纸箱城堡 / 树屋 / 写实童年照的红海,切到 80 年代末 OVA 怀旧动画风,直接跳出。**三次独立案例命中 → 升级为入库标准方法**。
