## 它控制什么 / 保留什么

### 视觉签名(sref 负责,稳定)

- **配色**:暖米白/象牙纸底 + 单一深色油墨。墨绿(deep teal/green)或纯黑两个分支,偶尔一点点品红/粉做单点提亮。
- **媒介质感**:凸版/木刻/油毡刻印感,边缘有压痕飞白、纸面颗粒噪点。
- **描边**:粗细不匀的手绘黑线,thin outline + 极少灰阶填充(大色块或纯线)。
- **构图**:常带一圈**手绘矩形外框**(招贴感);主体居中、留白大。

### 母题词汇(散文负责,发散)

这套组合反复吐出的物件,可当「它擅长画什么」的清单:

- **花**:雏菊 / 郁金香 / 万寿菊 / 松果菊,常画成**枯萎低垂**状(情绪滤镜结果)
- **早餐**:咖啡杯 / 茶壶 / 吐司 / 煎蛋
- **脸/头当容器**:头是花瓶、脸是面具、骷髅当花器(生死隐喻)
- **手**:捧脸 / 抱花束 / 按在头上
- **零件**:单只眼睛(茶壶上、额头上)、火焰、靴子、连衣裙人物

### 情绪基调

忧郁、温柔、不确定。即便不直接画悲伤,也偏向安静怅惘——花是蔫的,脸是低垂的,眼睛在流泪,骷髅=必死。

## 怎么用

1. **要一套「同款不同画」的招贴集**:保留这 4 个 sref + 风格后缀,只换主体散文 → 一次出 4 格,每格一个联想,风格统一。适合 zine / 专辑内页 / 情绪海报墙。
2. **想换题材但留版画质感**:把散文换成你自己的情绪文本(见 [[情绪散文当prompt主体_v1]]),sref 不动。
3. **想要黑色分支而非墨绿**:目前两色都会出,若要稳定单色,后缀补 `monochrome black ink` 一类锚词(待实测)。
