## 一句话定位

待实测。

本轮目标不是先给它贴标签,而是确认它作为一个独立"摄影师签名"的基础脾气:

- 它控制什么:颜色 / 主体 / 构图 / 情绪 / 材质 / 年龄 / 人种 / 伴生元素
- 它保留什么:prompt 的硬约束能不能守住
- 它加成什么:哪些题材一进去就变强
- 它排斥什么:哪些题材会被它重写或拖偏

## 第一轮实验纪律

### 固定条件

- 模型:`--v 8.1`
- 风格:`--raw`
- seed:`--seed 42`
- moodboard:关闭
- personalize/profile:关闭或保持默认,不要混入额外风格信号
- 每组只改 `--sw`,不要同时改 prompt

### sw 阶梯

每个题材跑三档:

```text
--sref 3234384125 --sw 100 --seed 42 --raw --v 8.1
--sref 3234384125 --sw 400 --seed 42 --raw --v 8.1
--sref 3234384125 --sw 800 --seed 42 --raw --v 8.1
```

观察重点:

| sw | 主要看什么 |
|---|---|
| 100 | 抽象风格层:色调 / 光线 / 肤质 / 氛围是否已渗透 |
| 400 | 中段是否不稳:pose / 构图 / 材质是否明显跳变 |
| 800 | 具象层:是否硬复刻主体原型 / 配饰 / UI / 字幕 / 特定物件 |
