## 三条行为规律(实测沉淀)

### 规律一 · 情绪覆盖律

强行重写以下维度,你写也白写:

| prompt 写什么 | sref 实际渲染 |
|---|---|
| `golden hour / warm light / sunset` | 阴天散射光 |
| `vibrant / saturated / dramatic` | muted ink-wash |
| `looking upward / smiling / serene` | 平视/略侧的克制凝视 |
| `pale rose lips / nude / glossy` | 深酒红 / 暗梅红哑光 |
| `romantic / cinematic dramatic` | 只剩 literary |

**保留**:角色五官、服装、配饰、季节符号、物件
**写法**:少写会被覆盖的词,多写会被加成的词(季节物件、同色系氛围、quiet/contemplative/still 状态词)

### 规律二 · 红色加重律

所有皮肤血色相关指令会被加重一档:
- `subtle blood flush` → 出明显血色
- `faint pink flush` → 出颊浮红
- `frostbitten flush` → 出**冻伤红块**(雷区!)

**写法**:用三层约束:
1. `the faintest hint of cold flush only on cheekbones`
2. `no nose flush`(反向命令)
3. `restrained pale tone`

### 规律三 · 年龄/成熟度覆盖律 ⭐

把 `woman` 强制渲染成 25-30 岁成熟女性,不接受少女:
- `young...in early twenties` → 仍偏成熟
- `youthful pristine porcelain` → 仍偏成熟
- `--sw 50` → 略年轻一档但仍不到位
- 改全套 prompt → 构图对了年龄仍不对

**结论**:**这个 sref 不能做少女向 IP**——硬边界。
