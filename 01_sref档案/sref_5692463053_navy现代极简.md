## 一句话定位

**Navy 蓝 + 米白 + 一抹红的现代极简平面美学。**

气质混合:Yohji Yamamoto 早期画册 / 90s 日本高级杂志广告 / 草间弥生剥光点点版 / Saul Leiter 彩色摄影裁掉一半色温。

## 色彩律(它最强的"独裁"维度)

- **配色铁律**:navy/cobalt + 米白/纯白 + 偶尔一抹猩红
- **修正版规则**:小面积暖色焦点都允许(红/橙黄/暖橘),但**大面积的暖**(夕阳、火光、整片金色稻田)会被吞掉
- **白热化替代**:题材带暖色冲突时,把暖色替换成"高对比白"(篝火变白火,金色 dawn 变冷光)
- **它无视用户的色彩描述**:鹦鹉绿→蓝,泥土褐→蓝,沙漠 dawn→蓝,白瓷碗→蓝

## 行为癖好

### 1. 二次创作癖

对极简概念会主动改剧本:
- "红蜡封 on 白纸" → 折成三角形的纸 + 一滴红蜡
- "红线穿过 navy 墙" → 红色电缆 + 配电箱 + 电线杆
- "户外茶道" → 自动加雪地
- "凤凰翅膀和尾巴是火" → 全身白光,只背部和鬃毛是火

### 2. 反密度 / 单数本能

```
1 人 → 完美   (kabuki / 灯塔小人 / 沙漠剪影)
2 人 → 顺滑   (背对背 / 静默 / 距离)
群体 → 妥协   (拆成"分散的孤独")
欢腾 → 拒绝   (要么删人,要么冷冻情绪)
```

**核心人设**:**"人可以存在,但不能温暖。"**

### 3. 极端舒适区:凌晨三点的世界

任何"凌晨三点"题材它都是天才:深夜 24h 加油站 / 麦当劳 / 便利店 / 灯塔 / 空荡机场跑道。

**DNA**:Edward Hopper 的 Nighthawks / Roy Andersson 北欧荒诞剧 / Wim Wenders 柏林苍穹下。

商业品牌扔进去会被剥光商业气,只剩"夜里有人没人都开着灯的房子"那种气质。

## 引导词杠杆

**有效**:
- `pixel game art / indie game art / atmospheric game still` → 可靠 push 到像素 mode

**无效**:
- `cinematic photograph` 和 `flat poster illustration` 基本被无视,甚至会**互换身份**(摄影词出来插画,平面词出来照片)

**未测**:
- 艺术家具体名字(Hopper / Saul Bass / Issey Miyake editorial)的杠杆

**已测(2026-05-02 multi-sref 探索副会话补测)**:
- `--sw` 撬动强度:**强** —— sw=800 时它**完全反吃 prompt 的服装色**(prompt 写"dark gray with gold patterns",出来仍是 navy 蓝)

---

## ⚠️ 隐藏特征(2026-05-02 multi-sref 探索副会话发现)

这个 sref 不只是"风格容器",还会**偷偷带这些东西**:

1. **欧裔现代女性主体倾向** — 单 sref + sw=800 时,人物会被推成欧裔,即使 prompt 没要求
2. **UI 元素漏出** — 出现对话框气泡 / 字幕条 / 字幕等"训练池伴生元素"

**含义**:用这个 sref 时,如果 prompt **没强力指定主体**(比如没明确写"asian woman"),会被 sref 主导。

**实操**:
- 主体必须显式写 `east asian woman` 或类似
- 高 sw 时做好 UI 元素漏出的心理准备
- 想要"navy 极简风但人是亚洲人" → 用 multi-sref 配一个亚洲人脸主导的 sref(参见 [[multi_sref行为]] 规律 6)

## 适合的题材

✅ 单一焦点 + 大留白
✅ "凌晨三点的世界"系(夜店 / 加油站 / 麦当劳 / 便利店)
✅ 极简符号化(动物 + 物件 + 一滴红)
✅ Butoh / Kabuki 单人剪影
✅ 商业符号去神圣化

❌ 群体场景 / 欢腾情绪
❌ 大面积暖色题材(夕阳 / 篝火 / 金色稻田)
❌ 古风水墨 / 「檐下」相关
❌ 复杂叙事场景

## 如何使用

1. **跑图前确认题材匹配**:不在"适合的题材"清单里,换 sref
2. **接受它的"独裁"**:别和它的色彩律对抗,顺它写
3. **巧用其"反密度"**:做"孤独/克制/凌晨三点"主题时它是利器
4. **暖色题材警告**:如果 prompt 里写 "warm / sunset / fire",做好它会改剧本的心理准备
