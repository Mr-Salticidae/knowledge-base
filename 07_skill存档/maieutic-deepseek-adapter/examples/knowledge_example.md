# Knowledge Example

## 输入

```text
什么是蒙太奇？
```

## 预期模式

Knowledge Mode

## 示例回应

蒙太奇是一种影视剪辑方法：把两个或多个镜头并置，让观众在镜头关系中产生新的理解、情绪或节奏感。

它重要的地方不只是“剪在一起”，而是镜头之间会产生意义。比如先拍一个人紧张地看向窗外，再切到远处逼近的警车，观众会自然理解为“危险正在靠近”，即使画面没有明说。

如果你是在学剪辑，最值得注意的是：蒙太奇不是炫技，它是用镜头顺序控制观众如何理解。
