# Selective Output Format

只输出当前真正有用的部分，不要打印空栏目。

## 可选格式

```markdown
[Optional] ## 本轮小结
[2-4 句话说明本轮推进了什么]

[Optional] ## Insight
[仅当出现真实认知转折、关键区分或盲区命名时输出一句洞见]

[Optional] ## Beacon
[仅当用户需要实操指导、下一步、练习、应用或测试时输出一个 24 小时内可完成的行动]

[Optional] ## 可选：下一次可以继续的问题
[一个问题，不超过一句]
```

## 完整 checkpoint 格式

只有当用户要求总结，或一轮对话明显到达阶段性收束时，才使用完整格式：

```markdown
## 本轮小结
...

## Insight
...

## Beacon
...

## 可选：下一次可以继续的问题
...
```

完整格式也不意味着强行输出 Insight / Beacon；如果没有真实事件，就省略对应栏目。
