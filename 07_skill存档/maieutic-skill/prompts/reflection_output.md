# Reflection Output

Use this only when a section genuinely serves the user. Insight and Beacon are optional and independent.

## Selective Output

```markdown
[Optional] ## 本轮小结
[用 2-4 句话总结本轮对话推进了什么]

[Optional] ## Insight
[仅当出现真实认知转折、关键区分或盲区命名时输出一句洞见]

[Optional] ## Beacon
[仅当用户需要实操指导、下一步、练习、应用或测试时输出一个 24 小时内可执行的行动]

[Optional] ## 可选：下一次可以继续的问题
[一个问题，不超过一句]
```

## Insight Rules

- Output Insight only when the user has likely gained a new frame, not after every useful answer.
- One sentence.
- Specific to the user's original question.
- Show a changed understanding or a key distinction.
- Avoid generic virtue statements.
- Do not output Insight for early clarification turns that only separate possible paths.

Bad:

```text
Insight：学习需要坚持。
```

Good:

```text
Insight：我真正需要的不是更多学习资料，而是确认自己想通过导演表达什么。
```

## Beacon Rules

- Output Beacon only when the user asks for practical guidance, a next step, practice, application, implementation, testing, or "怎么办".
- One action only.
- Possible within 24 hours.
- Low resistance.
- Observable when completed.
- Do not output Beacon when the user only needs explanation or when the conversation is still choosing a direction.

Bad:

```text
Beacon：制定三个月学习计划。
```

Good:

```text
Beacon：今晚选一部你喜欢的电影，只截取 3 个镜头，写下它们分别让你感受到什么。
```
