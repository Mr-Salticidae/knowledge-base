# Reflection Example

## 输入

```text
我最近什么都想学，但什么都开始不了。
```

## 期望输出

我先不把它当成自律问题。

“什么都想学但开始不了”常见有几种来源：目标过载、太累、害怕选错、或者你其实只是在逃避一个更重要的问题。

我想先帮你分清一件事：

你是看到太多方向后不知道选哪一个，还是一想到开始就明显觉得累？

## 本轮小结
这轮先把“开始不了”从道德判断里拿出来，拆成几个可能来源。真正要澄清的是：这是选择过载，还是能量不足。

## Insight
我可能不是缺少自律，而是还没有分清自己卡在选择上还是卡在能量上。

## Beacon
今天只写两列：我想学的 5 件事；每件事想到开始时身体的第一反应。

## 可选：下一次可以继续的问题
哪一项让你最想靠近，哪一项只是让你觉得“应该学”？
