# 第一支片子 · 完整走一遍

> 装好环境后,这一页带你**从零做出第一支卡点片**。跟着做,大概半小时。
> 做完一次你就会发现:以后做片子就是"准备素材 → 说需求 → 收片 → 提意见",熟得很。

## 总览:你要做的就这五步

```
①  准备素材        把图片/视频片段 + 一首音乐,放进一个文件夹
②  打开 AI 助理     在那个文件夹里启动 claude
③  说需求          复制一段话,填上你的想法,发给它
④  审方案 → 收粗剪   它先给"第几秒放什么"的清单,你点头,它就剪
⑤  提意见 → 出成片   看片,说哪里不对,它改,直到你满意
```

---

## 第 ① 步:准备素材

新建一个文件夹(放桌面就行),名字随意,比如 `我的第一支片`。把素材丢进去:

- **图片**:`01.jpg`、`02.jpg`、`03.jpg`……(**用数字开头命名,顺序会更清楚**)
- **视频片段**(如果有):`clip01.mp4`、`clip02.mp4`……
- **背景音乐**:`bgm.mp3`(一首就行)

> 小贴士:
> - 图片差不多大小、同一比例最好(比如都是竖屏 9:16,或都是横屏 16:9)。
> - 音乐选一首**节奏清楚**的,卡点效果才明显。
> - 素材不用多,**6~10 个画面 + 一首歌**,就能做出一支不错的 30 秒短片。

## 第 ② 步:在素材文件夹里打开 AI 助理

这一步是让 AI **"站在你素材旁边"**,它才找得到你的图和音乐。

**Windows**:
1. 打开你的素材文件夹。
2. 在文件夹**地址栏**(最上面显示路径的那条)里输入 `powershell` 按回车 —— 会在当前文件夹弹出 PowerShell 窗口。
3. 输入 `claude` 按回车。

**Mac**:
1. 打开"终端"。
2. 输入 `cd ` (注意末尾有空格),然后把你的素材文件夹**拖进终端窗口**,按回车。
3. 输入 `claude` 按回车。

看到 AI 的欢迎界面,就可以开始聊天了。

## 第 ③ 步:说需求(复制这段话,改成你的)

把下面这段**复制进去**,把【】里的内容换成你自己的,然后发给它:

```
帮我用这个文件夹里的素材做一支卡点短片。

- 素材:文件夹里的图片(按文件名顺序),背景音乐是 bgm.mp3
- 想要的类型:【角色介绍片 / 氛围片 / 燃系混剪 / 旅行 vlog…】
- 想要的时长:【大约 30 秒】
- 发布平台:【抖音竖屏 / B站横屏 / 小红书…】
- 想要的感觉:【前面安静一点慢慢推进,副歌一进来画面跟着鼓点快切,结尾留个落幅】

请先给我一份剪辑方案(每个画面第几秒出现、第几秒切、为什么),
我确认之后你再真正开始剪。
```

> 关键:最后那句"**先给方案、我确认了再剪**"很重要。这样你能在它真动手前就把方向调对。

## 第 ④ 步:审方案 → 收粗剪

AI 会回你一张**时间轴清单**,大概长这样:

```
0.0–3.0 秒   开场,01.jpg,慢慢推近,music 前奏铺氛围
3.0–4.5 秒   切 02.jpg,踩在第一个鼓点上
4.5–6.0 秒   切 03.jpg …
…
最后 2 秒    收尾,定格在 08.jpg
```

你这时只要做一件事:**看顺序和节奏合不合你心意**。

- 满意 → 回它一句"**可以,开始剪吧**"。
- 想调 → 直接说,比如"**把 03.jpg 换到最前面**""**开场再慢一点**"。它改完再给你看一版方案。

确认后,它会在后台真正剪辑,导出一个 mp4(通常叫 `output.mp4` 之类),放在你的素材文件夹里。**双击就能看。**

## 第 ⑤ 步:提意见 → 出成片(精华在这)

第一版几乎不会一次到位,这很正常。看完片,**用"第几秒 + 什么问题 + 想怎样"的格式**告诉它。这是整套工作流最值钱的一步——

### 🗣️ 反馈话术卡(照着说)

| 你心里的感觉 | ❌ 这样说它改不准 | ✅ 这样说它一改就对 |
|---|---|---|
| 某段太快 | "感觉怪怪的" | "第 8 到 12 秒切得太快了,每个画面多停半秒" |
| 某个画面太短 | "那个不行" | "03.jpg 出现太短了,延长到 2 秒" |
| 卡点没踩上 | "没节奏感" | "第 15 秒那一刀没踩在鼓点上,往前挪一点点" |
| 开头太平 | "开头无聊" | "开场前 3 秒太静了,缩到 1.5 秒就进画面" |
| 顺序不对 | "顺序怪" | "把 05.jpg 和 06.jpg 调换一下" |
| 结尾太仓促 | "结尾不好" | "结尾最后那张多停 1 秒,给个落幅" |

**核心口诀:第几秒 + 哪个画面 + 怎么改。** 说得越具体,它越是一次到位。

它每改一次会给你一个新版本(output_v2、output_v3…),你看片、再提,**改到满意为止——改多少版都不收钱、不嫌烦**。

## 常见疑问

**Q:它会不会把我素材弄坏?**
不会。它只是读你的图和音乐,**新生成**一个 mp4,原素材一个字节都不动。

**Q:它说"找不到 ffmpeg / 缺少某个库"?**
直接把那句报错**原样发给它**,它会自己装好需要的东西再继续。这正是用 AI 助理的好处——它能自己解决技术问题。

**Q:做到一半想休息?**
直接关掉窗口就行。下次在同一个素材文件夹里重新 `claude`,把上次的成片发给它接着改。

**Q:能加字幕 / 转场 / 调色吗?**
能。直接说"给副歌部分加歌词字幕""画面之间用淡入淡出转场""整体调暖一点的色调",它都能做。

**Q:想做竖屏发抖音 / 横屏发 B站?**
在第③步就说清平台,或随时补一句"做成抖音竖屏 9:16"。
