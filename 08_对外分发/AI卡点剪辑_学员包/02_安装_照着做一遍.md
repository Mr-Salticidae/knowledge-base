# 安装 · 照着做一遍(只需做一次)

> 这一步是把"AI 剪辑助理"请到你电脑上。听起来吓人,其实就是**双击一个文件 + 复制粘贴一行字**。
> 一辈子只用做一次,以后做片子直接跳到 03。

## 开始之前:你需要一个账号(说在前面)

AI 剪辑助理叫 **Claude Code**,它背后是 AI,**需要一个 Claude 会员账号**才能用(就像视频网站要会员才能看片)。

- 去 **claude.ai** 注册一个账号。
- 个人用,开 **Claude Pro**(月付会员)就够做片子了。
- 注册和付费在网页上完成,跟开任何会员一样,这里不展开。

> **名词小抄**
> - **Claude Code**:能在你电脑上动手干活(写代码、剪视频)的 AI 助理。
> - **FFmpeg**:真正执行剪辑的引擎,没有它剪不了片。一键脚本会帮你装。
> - **Python**:AI 写的剪辑代码靠它运行。一键脚本也会帮你装。
> - **终端 / PowerShell**:一个能打字给电脑下命令的黑框框窗口。你只会用到它输入一行 `claude`。

## 🪟 如果你用 Windows

### 第 1 步:解压

把这个学员包**整个文件夹解压出来**(不要在压缩包里直接双击)。放桌面就行。

### 第 2 步:双击一键脚本

进到解压出来的文件夹,找到 **`一键安装_Windows.bat`**,**双击它**。

- 会弹出一个黑色窗口,按提示按回车,然后**等它自己跑**。
- 中途如果弹"是否允许此应用更改你的设备",点 **是**。
- 它会依次装好 Claude Code、FFmpeg、Python,并自动放好"技能说明书"。

> 如果双击后窗口一闪而过 / 提示找不到 winget:说明你的系统组件需要更新。
> 打开"Microsoft Store" → 搜"应用安装程序" → 更新它,再重新双击脚本。

### 第 3 步:登录(让 AI 认识你)

脚本跑完后:

1. 在开始菜单搜 **PowerShell** 或 **Windows Terminal**,打开它。
2. 输入这一行,按回车:
   ```
   claude
   ```
3. 它会自动打开浏览器,用你第 0 步注册的账号登录,按提示点完成。

看到欢迎界面,就装好了 ✅

### 万一一键脚本不灵(手动兜底)

打开 PowerShell,把下面**逐行**复制进去、各按一次回车:

```powershell
winget install --id Anthropic.ClaudeCode -e
winget install --id Gyan.FFmpeg -e
winget install --id Python.Python.3.12 -e
```

技能说明书手动安装(把学员包里的 `skills\blind-editing-workflow` 文件夹,复制到你用户目录下的 `.claude\skills\` 里即可)。实在搞不定,把这页发给懂行的朋友,三分钟的事。

## 🍎 如果你用 Mac

### 第 1 步:解压

把学员包**整个文件夹解压**出来,放桌面。

### 第 2 步:双击一键脚本

找到 **`一键安装_Mac.command`**,**双击它**。

- 第一次双击,Mac 可能拦一下(提示"无法打开,因为来自身份不明的开发者")。
  解决:**右键点这个文件 → 选"打开" → 在弹窗里再点"打开"**。这样就放行了。
- 如果提示没有运行权限,打开"终端",输入 `chmod +x ` (注意末尾有空格),然后把那个 `.command` 文件**拖进终端窗口**,按回车;之后再双击就能跑。
- 跑的过程中可能让你**输入开机密码**(装 Homebrew 时),输入时屏幕不显示字符是正常的,打完按回车即可。

它会自动装好 FFmpeg、Python、Claude Code,并放好技能说明书。

### 第 3 步:登录(让 AI 认识你)

1. 打开**终端**(在"启动台"里搜"终端"或"Terminal")。
2. 输入这一行,按回车:
   ```
   claude
   ```
3. 浏览器会自动打开让你登录,用第 0 步的账号登入即可。

看到欢迎界面,就装好了 ✅

### 万一一键脚本不灵(手动兜底)

打开终端,逐行粘贴执行:

```bash
# 如果没装过 Homebrew,先装它:
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 再装三件套:
brew install ffmpeg python
brew install --cask claude-code
```

技能说明书手动安装:把学员包里的 `skills/blind-editing-workflow` 文件夹,复制到你个人目录下的 `.claude/skills/` 里。

## 怎么确认装好了(自检)

打开终端 / PowerShell,**逐行**输入下面三条,只要每条都能打印出一段文字(而不是"找不到命令"),就说明装好了:

```
claude --version
ffmpeg -version
python --version
```

> Windows 上如果 `python` 不灵,试试 `py --version`。
> 如果刚装完提示"找不到命令",把终端/PowerShell **关掉重开**一次再试(新装的程序需要重开窗口才认得)。

## 会咬人的坑(提前知道,少走弯路)

1. **一定要先解压整个文件夹再双击脚本**。在压缩包里直接双击,脚本找不到旁边的 `skills` 文件夹,技能装不上。
2. **装完记得关掉终端重开一次**。新装的 `claude`/`ffmpeg`/`python` 命令,旧窗口可能还不认识,重开就好。
3. **Claude Code 要会员账号**。没登录会让你登录,跟着浏览器走完就行。
4. **公司电脑 / 受管控的电脑**可能禁止自动安装。如果脚本一直失败,大概率是权限被限制,换台个人电脑,或找 IT/朋友帮忙。
