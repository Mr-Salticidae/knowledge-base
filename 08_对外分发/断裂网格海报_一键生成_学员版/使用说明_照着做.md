# 使用说明 · 照着做一遍

给完全没碰过代码的朋友。跟着做，十分钟出第一张海报。

---

## 第一步：装 Python（只做一次）

1. 打开网址：https://www.python.org/downloads/
2. 点黄色大按钮 **Download Python**，下载完双击安装包。
3. **最关键**：安装第一屏，勾选最下面那个 **Add Python to PATH**，再点 Install Now。
4. 装完点 Close。这一步一辈子只做一次。

> 怎么确认装好了？按 `Win + R`，输入 `cmd` 回车，在黑窗口打 `python --version`，
> 能显示 `Python 3.xx` 就成了。

---

## 第二步：出图（每次都这么用）

1. 把这个文件夹整个放到桌面（别只拿一个文件出来，几个文件要在一起）。
2. 把你想做海报的图片，用鼠标**拖到** `把图片拖到我身上_Windows.bat` 图标上，松手。
3. 弹出黑窗口，第一次运行会自动装一个图像库（联网、十几秒），之后就不用了。
4. 看到 `✅ 完成！` 就好了。成品 PNG 存在**你原图的同一个文件夹**里，
   名字是「原名_海报.png」。

就这么简单。不想拖？把图片和 `海报生成器.py` 放同一个文件夹，双击 .py 也行——
它会自动挑文件夹里最新那张图。

---

## 第三步：改成你自己的文字（可选）

右键 `海报生成器.py` → 用**记事本**打开。只动最上面这一段，别的别碰：

```python
# ============ 改这里就够了 ============
TITLE   = "STATIC / BLOOM"     # 大标题
BODY    = ("在这里写正文，多长都行，程序会自动折行……")   # 正文
AUTHOR  = "your name"          # 署名
DATE    = "Jul 2026"           # 日期
NUMBER  = "N°01"               # 右上角小编号
THEME   = "暗"                 # 填 "暗" 或 "亮"
# ====================================
```

改完 `Ctrl + S` 保存，再拖一次图片就是新文字了。

- `THEME` 填 `"暗"` 是墨绿暗调，填 `"亮"` 是奶粉亮调，两种都试试挑喜欢的。
- 标题用英文更有杂志味；想用中文也可以，正常打字即可。

---

## 出错了怎么办

| 现象 | 解决 |
|---|---|
| 双击 .bat 闪一下就没了 / 提示没找到 Python | Python 没装，或装时没勾「Add Python to PATH」。重装一遍，记得勾。 |
| 提示 `❌ 出错了` | 多半是图片路径有特殊字符，或图片坏了。把图片改个简单英文名（如 `a.jpg`）再拖。 |
| 成品里主体被切到头 / 缺脚 | 换一张更竖的图（3:4 左右），主体别顶到边。 |
| 标题是中文但显示成方块 | 你把 TITLE 换成中文了，但默认用的是英文字体。中文标题保持能显示的字体即可，或标题仍用英文。 |
| 第一次运行卡在装 Pillow | 需要联网。等十几秒；实在装不上，在 cmd 里手动打 `pip install Pillow`。 |

## 想要的效果和现在差很多？

这套是「一张竖图 → 断裂网格杂志海报」的固定模板，改的是文字和配色。
如果你想换整体版式（比如格子排布、装饰样式），那就是改代码本身的活了，
可以把需求发给你的技术搭子——脚本结构很清晰，改起来不难。
