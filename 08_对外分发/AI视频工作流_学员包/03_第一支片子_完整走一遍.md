# 第一支片子 · 完整走一遍

> 装好环境后,这一页带你**从一个想法做出第一支成片**。跟着做,大概一小时(第一次慢,熟了很快)。
> 做完一次你就会发现:以后做片子就是"说想法 → 按提示备素材 → 看片提意见",越用越顺。

## 先挑个好上手的第一支

强烈建议第一支做 **「一支 30–60 秒的图形科普片 / 讲解片」**,理由:
- 画面用**代码画的图形动画**(总指挥直接搞定),**第一支不用请画师 MJ**,省钱省事。
- 只需要**核心三件套**(总指挥 + 配音 + 配乐),流程最短,最容易跑通。
- 跑通一次,再做"配真实图片的 MV""角色 PV""批量短视频"就都是同一套路、加几个工具的事。

> 想第一支就做音乐 MV 或带真实图片的片也行,流程一样,只是第③步多一步"去 MJ 出图"。本页用科普片举例。

## 总览:你要做的就这六步

```
①  开对话         在一个空文件夹里启动总指挥 claude
②  说想法         复制一段话,填上你的选题,发给它
③  审脚本+方案     它先给脚本、分镜、配音稿,你点头(还没真做)
④  备素材         按它的提示:配音(ElevenLabs)+ 配乐(Suno),拿回来给它
⑤  拼成片         它用剪辑台 Remotion 把画面+配音+字幕+音乐拼出 mp4
⑥  提意见→出成片   看片,说哪里不对,它改,直到满意
```

---

## 第 ① 步:开一个对话(给总指挥一块工作台)

新建一个空文件夹(放桌面),比如 `我的第一支片`。让总指挥"站"在这个文件夹里,它的成果都会放这儿。

**Windows**:打开这个文件夹 → 在最上面的**地址栏**输入 `powershell` 按回车 → 在弹出的窗口输入 `claude` 按回车。
**Mac**:打开"终端" → 输入 `cd ` (末尾空格)→ 把文件夹**拖进终端**按回车 → 输入 `claude` 按回车。

看到 AI 欢迎界面,就能开聊了。

## 第 ② 步:说想法(复制这段话,改成你的)

把下面这段**复制进去**,把【】换成你自己的,发给它:

```
我想做一支知识科普 / 讲解短视频,我不会剪辑也不会写代码,
请你当总指挥,用蒙眼剪辑法 + Remotion 帮我做出来。

- 选题:【用一分钟讲清楚"复利"是怎么回事】
- 时长:【大约 45 秒】
- 风格:【轻松、扁平图形动画那种,像“一分钟讲懂”系列】
- 平台:【抖音竖屏 9:16 / B站横屏 16:9 选一个】
- 配音:要中文旁白(我会用 ElevenLabs 生成)
- 配乐:要一段轻快 BGM(我会用 Suno 生成)

请按这个顺序来,每一步先给我看、我确认了再往下:
1. 先写脚本和旁白稿(分几个段落、每段讲什么、旁白原话)
2. 再列分镜方案(第几秒画面是什么)
3. 然后告诉我去哪些工具、用什么提示词生成配音和配乐
4. 素材齐了,你用 Remotion 拼成片
```

> 关键:最后那句"**每一步先给我看、确认了再往下**"很重要。它会激活技能乐队、按蒙眼剪辑法一步步来,而不是闷头把整片做完。

## 第 ③ 步:审脚本 + 分镜(还没真做,放心改)

总指挥会先给你**脚本 + 旁白稿 + 分镜表**,大概长这样:

```
段落1(0-6s)钩子:旁白“你有没有想过,一块钱怎么变成一座金山?”
                画面:一枚硬币滚动,慢慢堆成小山
段落2(6-20s)原理:旁白“秘密就是复利……”
                画面:雪球越滚越大的图形动画
…
结尾(40-45s)总结 + 一句话 CTA
```

你这时只做一件事:**看脚本顺不顺、对不对你的意**。
- 满意 → "可以,继续"。
- 想改 → 直接说,"开头钩子换一个更扎心的""第二段太啰嗦,砍一半"。它改完再给你看。

> 脚本是地基,**在这步多花点时间值得**——脚本对了,后面才不会白做素材。

## 第 ④ 步:备素材(你跑腿,它指挥)

脚本定了,总指挥会**给你明确的"采购清单"**,通常是:

**配音**:它把每段旁白的文字 + 推荐的声音风格给你,你照做——
1. 打开 ElevenLabs(或 Fish Audio),选个中文声音;
2. 把旁白文字贴进去,生成,下载音频;
3. 把下载的音频文件**拖进你那个项目文件夹**,回总指挥一句"配音放好了,文件名叫 xxx"。

**配乐**:它把一段 Suno 提示词(描述风格/节奏/情绪)给你,你照做——
1. 打开 Suno,把提示词贴进去生成;
2. 挑一首满意的下载,拖进项目文件夹;
3. 回它"BGM 放好了"。

> - **字幕你不用管**:配音放好后,总指挥会在你电脑上自动跑"字幕员"(Whisper),把配音转成逐字字幕。
> - **画面也不用管**(科普片):图形动画总指挥用代码直接画。
> - 如果做的是**真实图片的片**:这一步它还会给你 MJ 提示词,你去 Midjourney 出图、拿回来即可。

## 第 ⑤ 步:拼成片

素材齐了,跟它说"**素材都齐了,开始拼吧**"。它会在后台用剪辑台(Remotion)把画面、配音、字幕、音乐精确拼好,导出一个 mp4 放进你的文件夹。**双击就能看。**

> 第一次它可能要花几分钟建工程、装东西、渲染,属正常。中间若它说"缺某个组件",不用慌——它会自己装好继续。

## 第 ⑥ 步:提意见 → 出成片(精华在这)

第一版几乎不会一次到位,正常。看完片,**用"第几秒 + 什么问题 + 想怎样"的格式**告诉它。这是整套工作流最值钱的一步——

### 🗣️ 反馈话术卡(照着说)

| 你心里的感觉 | ❌ 这样说它改不准 | ✅ 这样说它一改就对 |
|---|---|---|
| 某段太快/太慢 | "感觉怪怪的" | "第 8 到 12 秒太快了,每个画面多停半秒" |
| 配音和画面没对上 | "不同步" | "第 15 秒旁白说到‘复利’时,雪球画面才该出现" |
| 字幕位置不好 | "字幕难看" | "字幕太靠下被进度条挡了,往上挪到画面中间偏下" |
| 开头太平 | "开头无聊" | "前 3 秒钩子不够狠,换成‘90% 的人都算错了这笔账’" |
| 某个画面要换 | "那个不行" | "第二段那个图形太花,换成简单的雪球滚动" |
| 音乐太吵 | "BGM 大了" | "BGM 压到旁白音量的三分之一,别盖住人声" |
| 结尾太仓促 | "结尾不好" | "结尾总结那句多停 1.5 秒,给个落幅" |

**核心口诀:第几秒 + 哪个东西 + 怎么改。** 说得越具体,它越是一次到位。

它每改一次给你一个新版本,你看片、再提,**改到满意为止——改多少版都不收钱、不嫌烦**。

## 进阶玩法(第一支跑通后再看)

- **做音乐 MV / 卡点混剪**:把"科普片"换成"卡点片",素材换成你的图片/视频 + 一首歌,告诉它"画面踩着鼓点切"。
- **做角色介绍片(PV)**:多个角色逐一登场,每个配一句独白 + 设定展示。要角色跨画面长一样,告诉它"用 MJ 角色一致性那套",它会引导。
- **一次批量出几十上百条**:适合"同一个模板套不同内容"的场景(比如一批知识点、一批案例)。先精修**一条母版**到满意,再让它"套这批数据批量出片"。这就是新版最强的地方——**改一次母版,代码批量复制**。
- **分发多平台**:同一条竖版母版,抖音/B站/快手通用,差别只在封面、标题、标签。告诉它平台,它能帮你出各平台的封面和文案建议。

## 常见疑问

**Q:我电脑上什么都没有,它真能自己建好视频工程?**
能。总指挥会用一条命令(`npx create-video`)新建一个 Remotion 工程,把素材接进去。你不用懂,这是它的活。

**Q:它说"缺 xxx / 报错了"?**
把那句报错**原样发给它**,它会自己解决。这正是用 AI 总指挥的好处。

**Q:做到一半想休息?**
关掉窗口即可。下次在同一个项目文件夹重新 `claude`,跟它说"接着上次的做",它能继续。

**Q:必须把所有工具都用上吗?**
不必。最少**总指挥 + 配音 + 配乐**就能做图形科普片;想要真实画面再加画师(MJ)、动画师(Seedance)。按需请人。

**Q:钱花在哪?**
主要是几个工具的会员费(Claude、配音、作曲,想要画面再加 MJ)。先用核心三件套试水,跑通再决定加哪些。
