# 安装 · 照着做一遍(只需做一次)

> 这一步是把"总指挥 AI + 它的工具台"请到你电脑上。听起来吓人,其实大头就是**双击一个文件等它跑完**。
> 一辈子只用做一次,以后做片子直接跳到 03。

安装分两部分:
- **A. 装电脑上的"地基"**(总指挥 + 剪辑台运行环境)——有一键脚本,基本自动。
- **B. 注册几个"AI 工具账号"**(画师、配音、作曲)——这些是网上的在线服务,得你自己注册,像开会员一样。

---

## A 部分:装地基(一键脚本)

### 开始之前:总指挥需要一个账号

总指挥 AI 叫 **Claude Code**,它背后是 AI,**需要一个 Claude 会员账号**才能用(就像视频网站要会员)。
- 去 **claude.ai** 注册账号,个人用开 **Claude Pro**(月付)就够。
- 注册和付费在网页完成,跟开任何会员一样。

> **名词小抄**
> - **Claude Code**:总指挥,在你电脑上动手干活的 AI。
> - **Node.js**:剪辑台 Remotion 运行需要的"底座"程序。一键脚本帮你装。
> - **FFmpeg / Python**:处理音视频、跑字幕等技术活要用的工具。一键脚本帮你装。
> - **终端 / PowerShell**:一个能打字给电脑下命令的黑框框窗口。你主要只会用它输入 `claude`。

### 🪟 如果你用 Windows

1. **解压**:把这个学员包**整个文件夹解压**出来(别在压缩包里直接双击),放桌面。
2. **双击**文件夹里的 **`一键安装_Windows.bat`**。
   - 弹出黑窗口,按提示按回车,然后**等它自己跑**(会装好几样东西,耐心等)。
   - 中途弹"是否允许更改设备",点 **是**。
   - 它会依次装好:Claude Code、Node.js、FFmpeg、Python,并把整支"技能乐队"放到位。
3. **登录**:跑完后,开始菜单搜 **PowerShell** 打开,输入下面这行按回车:
   ```
   claude
   ```
   会自动打开浏览器登录,用你刚注册的账号登入即可。看到欢迎界面就成了 ✅

> 双击后一闪而过 / 提示找不到 winget?→ 打开"Microsoft Store" → 搜"应用安装程序" → 更新它,再重新双击脚本。

### 🍎 如果你用 Mac

1. **解压**整个文件夹,放桌面。
2. **双击** **`一键安装_Mac.command`**。
   - 第一次双击若被拦("来自身份不明的开发者"):**右键点它 → 选"打开" → 再点"打开"**。
   - 若提示没运行权限:打开"终端",输入 `chmod +x ` (末尾有空格),把那个 `.command` 文件**拖进终端**按回车,之后再双击。
   - 跑的过程中可能让你**输入开机密码**(装底座时),输入时屏幕不显示是正常的。
   - 它会装好:Homebrew、Node.js、FFmpeg、Python、Claude Code,并放好技能乐队。
3. **登录**:打开"终端",输入 `claude` 按回车,浏览器登录即可 ✅

### 手动兜底(一键脚本不灵时)

**Windows**(PowerShell 里逐行粘贴执行):
```powershell
winget install --id Anthropic.ClaudeCode -e
winget install --id OpenJS.NodeJS.LTS -e
winget install --id Gyan.FFmpeg -e
winget install --id Python.Python.3.12 -e
```
**Mac**(终端里逐行执行):
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install node ffmpeg python
brew install --cask claude-code
```
技能乐队手动安装:把学员包里 `skills` 文件夹**里面的每一个子文件夹**,复制到你个人目录下的 `.claude/skills/` 里(Windows 是 `%USERPROFILE%\.claude\skills\`)。实在搞不定,把这页发给懂行的朋友,几分钟的事。

### 确认地基装好了(自检)

打开终端 / PowerShell,逐行输入,每条都能打印出文字(而不是"找不到命令")就对了:
```
claude --version
node --version
ffmpeg -version
python --version
```
> Windows 上 `python` 不灵就试 `py --version`。刚装完提示找不到命令?把窗口**关掉重开**再试。

## B 部分:注册 AI 工具账号(乐队成员)

地基只是把"总指挥"和"剪辑台"装好了。真正出片还要请几位工具帮忙。**它们都是网页/App,在线注册即可,不用装到电脑里。**

按"先必需、后可选"分三档,**新手建议先只搞定第 1 档就开工**:

### 第 1 档 · 核心三件套(强烈建议先备齐,做出第一支片就够)

| 成员 | 是谁 | 去哪 | 作用 |
|---|---|---|---|
| 总指挥 | Claude Code | claude.ai 注册(A 部分已做) | 统筹 + 出片 |
| 配音演员 | **ElevenLabs** 或 **Fish Audio** | 各自官网注册 | 把旁白读成真人声 |
| 作曲家 | **Suno** | suno.com 注册 | 量身定做 BGM |

> 字幕员(Whisper)和剪辑台(Remotion)不用注册——总指挥在你电脑上自动跑。

### 第 2 档 · 让画面更高级(想做有画面的片再加)

| 成员 | 是谁 | 作用 |
|---|---|---|
| 画师 | **Midjourney(MJ)** | 画每个画面的关键图 |
| 动画师 | **Seedance / 即梦** | 让静态图动起来(图生视频) |

### 第 3 档 · 锦上添花(以后再说)

- 角色要跨多个画面长得一样 → 用到 MJ 的"角色一致性"玩法(技能乐队里已带说明书,总指挥会引导)。
- 做纯图形的科普/讲解片(类似"在一分钟内讲懂")→ 总指挥可以直接用代码画图形动画,**连画师都不一定需要**。

> 钱要花多少?这些工具大多是月付会员,几十到一两百元/月不等,可按需开。**只想先试试水**:先用核心三件套,甚至先做一支"纯图形科普片"(只用总指挥 + 剪辑台,几乎零额外成本),跑通流程再决定要不要请画师作曲家。

## 会咬人的坑(提前知道,少走弯路)

1. **先解压整个文件夹再双击脚本**。在压缩包里直接双击,脚本找不到旁边的 `skills`,技能装不上。
2. **装完关掉终端重开一次**。新装的 `claude`/`node`/`ffmpeg` 命令,旧窗口可能还不认识,重开就好。
3. **核心账号要会员**。Claude、配音、作曲那几位多数要付费,这是这套工作流的"硬件成本",省不掉。
4. **公司/受管控电脑**可能禁止自动安装。脚本一直失败大概率是权限受限,换个人电脑或找人帮忙。
5. **不用一次集齐所有工具**。先核心三件套,缺画师时总指挥会提醒你"该去请画师了",到时再注册不迟。
