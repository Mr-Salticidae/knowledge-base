# 目标是成为 Prompt 大师

一个面向 AIGC 小白的 prompt 拆解系列。

每一期挑一个 prompt battle 主题,讲清楚四件事:
**难在哪 → 怎么破题 → 完整 prompt → 四层拆解与可复用的思路。**

每篇笔记都可**单独阅读**,不依赖其它文档;配套有可直接发布的小红书卡片(3:4 竖版)和小红书正文。

## 每期文件夹里有什么

- `XX_笔记.md` —— 独立可读的拆解笔记(小白向,长 · 存档)
- `XX_小红书正文.md` —— 可直接发布的小红书正文(短 · 内敛克制)
- `XX_封面卡.png` —— 小红书封面图
- `XX_prompt拆解卡.png` —— prompt 四层标注图
- `XX_原图.png` —— 作品原图
