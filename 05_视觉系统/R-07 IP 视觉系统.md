# R-07 IP · 视觉系统

> 建立:2026-05-10 (R-07 项目主对话 + MJ 出图阶段)
> 锁定:2026-05-11 (17 张 MJ final + 17 段可灵视频成片)
> 评级:A / A / A-(全 17 张人物一致性 / 风格一致性 / 剧本契合度)

## 概念

**R-07** = 一台被遗忘的旧时代机体,在人类文明退场后的废墟里,用一段被偶然植入的音频生成模型,对着任何还活着的东西唱歌。

核心反差:**无机的造物在为有机的生命歌唱**。

## 视觉识别符四要素(必须全部满足)

| # | 要素 | 规范 |
|---|---|---|
| 1 | **面甲** | **红色圆盘(上半部)+ 灰白机械纹理(下半部)** 的混合结构,**不是纯红整面** |
| 2 | **传感器** | **单个红色发光点**,位于面甲**右下方的灰白机械纹理区域**,**不在中央** |
| 3 | **机体** | **哑光黑主导**(60-70%)+ **红色装甲**分布在**肩部 / 胸甲 / 上背**(30%) |
| 4 | **颈缆** | **多束颈部线缆束**,从颈后/侧面伸出 |
| - | 比例 | humanoid 标准,纤瘦但正常,有关节细节 |

## 配色规范

### 主调
- **冷灰雾蓝**(70%+)
- **哑光黑**机体
- **油画质感** + 35mm 胶片颗粒

### 暖色锚点(极少量)
- R-07 红装甲 / 铁锈红 / 帝王蝶橙翅 / 野花黄 / 兔毛 / 猫毛
- **低饱和**,呈现"油画自然色"而非"鲜艳数字色"
- 暖色锚点必须**少而精**,Outro 段递减到几乎不可见

### 配色禁区
- ❌ 赛博朋克霓虹(紫粉蓝霓虹灯 / 全息广告)
- ❌ 糖水二次元
- ❌ 高饱和数字色

## 行为禁区

R-07 是机器,所有行为表达必须**机械 / 克制 / 目的性弱**:
- ❌ 流泪 / 任何"哭"的拟人表达
- ❌ 救人类 / 跟人类有交集(人类已退场)
- ❌ 自指 AI 身份("我是 AI / 我没有心")
- ❌ 拟人化情感外露(摆姿势 / 比心 / 招手等)
- ❌ 表演 / 跳舞 / 唱歌时的"舞台姿态"

允许:
- ✅ 缓慢转头(机械感)
- ✅ 抬手 / 举手(机械关节铰链感)
- ✅ 行走(脚步分明,机器节奏)
- ✅ 完全静止(凝固)

## 情绪基调

**"安静的剩余"**,不是"悲伤的末日"。

R-07 不知道自己是悲剧还是喜剧。它身上没有怨气、没有控诉、没有挣扎——这种克制反而是最打动人的地方。

参考:
- 宫崎骏《天空之城》最后那个守护花园的机器人
- 《瓦力》前 30 分钟没有台词的部分
- Radiohead 那种克制的悲伤,但更柔软

## 世界观锚点

- 时间:不确定的未来,人类已退场很久
- 地点:废弃工业区 / 数据中心 + 周边自然回收的土地(苔藓 / 藤蔓 / 杂草 / 野花)
- 反差:**无机的造物在为有机的生命歌唱**
- 受众:R-07 没有受众反馈 / 没有播放量 / 没有评论。它只在每次遇到"还活着的东西"时,停下来,发出一段声音

## 元叙事张力

**R-07 不知道自己是偶像。** 在一个人人都想红的活动里,放一个根本不知道自己在被观看的角色——它不在表演给你看,你只是恰好路过它。

## 如何使用

### 后续 R-07 衍生创作
- 任何 R-07 出图必须 cref/oref 引用 S05 锚点图
- 视觉识别符四要素逐张自检
- 行为禁区严格遵守(尤其不要被生成式 AI 默认的"机娘 / 萌化"倾向影响)

### 跨项目借鉴
- "无机造物为有机生命歌唱"的结构 → 可以泛化为"非中心叙事"(把传统故事的主体放在边缘)
- "不展示参赛标识"的克制 → 见 [[R-07创作内核_克制微动_天真文案_减法原则|R-07 创作内核：克制、天真与减法]] § 三 违和论

### 不要做的事
- ❌ 把 R-07 改成"更可爱" / "更主流"(任何"加一点可爱元素 / 加高光颜色"的调整都是反方向)
- ❌ 把 R-07 跟 NULL-07 混淆(NULL-07 是另一个独立 IP,排除在 R-07 项目外)
- ❌ 在 R-07 故事里加入人类角色
