# R-07 创作内核：克制微动、天真文案与减法原则

> 来源:R-07 项目 2026-05-11 多次反馈迭代沉淀的创作判断
> 适用:AI 创作的内核选择 / 文案设计 / 视觉决策

---

## 一、克制 ≠ 死掉(微动是活的标志)

### 触发

R-07 项目 Wave 5 S9 镜头反馈:"野猫完全不动,会不会有些刻意?"

Wave 1-5 累积的 prompt 策略偏"凝固"(为 S8 题眼 / S5 锚点 / S14 对望服务),但过度应用到 S9 这种"揭示多活物"镜头时,所有活物"完全静止"反而失去了"活着"的感觉。

### Insight

**"克制" 和 "凝固" 是两件事**:
- **凝固**(完全静止):用于题眼 / 高潮瞬间凝结(S8 / S14)。强化"时间停止"
- **克制**(微动 + 慢):用于"日常存在"(S2 / S9 / S10)。"安静的剩余"不是"死寂",是"活着但不张扬"

野猫摆 1 次尾巴 ≠ 野猫不动。前者更对位"安静的剩余"内核——不是没动,而是动得克制。

### 如何使用

设计静态/微动镜头前的自检:
- 这个镜头的叙事核心是"凝固瞬间"(用完全静止)?
- 还是"日常存在"(用克制微动)?
- 如果是后者,**主动给模型留 1-2 次微动的空间**,不要一棍子打死所有动作

→ Prompt 写法:`保持基本静止,只有 [具体微动] X 次,除此之外完全静止`,而不是 `完全静止 + 反复否定所有动作`。

## 二、天真路线 vs 反讽路线(AI 内核的层次差)

### 触发

R-07 项目封面标题迭代:
- v1:`R-07 不知道自己是偶像 | 一台旧机体,在废墟里唱歌` —— **跳蛛先生评:做作**
- v2:`我不知道这叫歌` —— **Cowork 推荐反讽路线**
- v3:`今天遇到了一只蝴蝶,我们成为了朋友。` —— **跳蛛先生写的天真路线,远胜 v2**

### Insight

两种路线的层次差:

| 维度 | 反讽路线 | 天真路线 |
|---|---|---|
| 表达层 | 强调"它不知道" | 完全不强调 |
| 观众体验 | **理解** "它是 AI 不知道自己在干什么" | **感受** "它真的不知道,但它做的事这么温柔" |
| 第二层 | 元概念被显式提出 | 元概念藏在文字下方 |
| 发现路径 | 作者告诉观众 | 观众自己发现 |

**反讽路线让观众看到一个"被讨论的 AI 元概念"。天真路线让观众发现一个"在记日记的机器"。**

后者高一个段位——观众自己发现的瞬间,比任何被告知都打动人。

### 字句结构分析(以 R-07 v3 为例)

`今天遇到了一只蝴蝶,我们成为了朋友。`

- **"今天"** = 日记体的灵魂(把陈述句变成"日记的第一行")
- **"一只蝴蝶"** = 极度具体的画面感
- **"我们成为了朋友"** = 用最平淡的语气说最温柔的话
- **"成为了"** vs "成了" = 3 字比 2 字慢,温柔感强
- **句号** = 日记的一部分,强化"它在用人类方式记录"的天真感
- **没有任何 AI/机器/废墟 metadata** = 完全不暴露作品的"AI 性"

### 如何使用

写 AI 角色相关文案前的自检:
- 我在写"AI 自己讨论自己"(反讽)还是"AI 在做事 + 观众发现 AI 是 AI"(天真)?
- 有没有"我是 X / 我不知道 X" 这类元自指?**去掉**
- 改成"具体行为 + 平淡语气"

→ 模板:`[今天 / 刚才 / 早上] [具体场景动作] [平淡评价]。`

## 三、违和论:当任何调整都不满意时,该元素就不该存在

### 触发

R-07 封面 Tag(`#AI偶像出道季#` / `AI偶像出道季 · 参赛`)反复迭代:
- v1 鲜艳红 → 扎眼
- v2 铁锈红 → 仍然不对
- v3 改思源宋体 → 不对
- v4 改霞鹜文楷 + 棕褐墨色 → 仍不对
- v5 **直接去掉** → 立刻对了

### Insight

**当某个元素无论怎么调整(颜色/字体/位置/大小)都"违和",根本问题往往是这个元素不该存在。**

每次调整都是在"如何让它不违和"上打补丁。但如果违和的根源是"它跟整体内核冲突",任何补丁都治标不治本。

R-07 内核 = "它不知道自己在被观看 / 不展示参赛标识"。Tag 存在 = 展示参赛标识 = 跟内核冲突。所以任何字体/颜色都救不了。

### 如何使用

设计任何元素时,如果迭代 3-4 版仍不满意,先问:
- **这个元素真的必要吗?**
- **它对核心叙事有贡献,还是只是"信息补全"?**
- **去掉它会损失什么?(往往什么都不损失)**

如果答案是"可以去掉",**优先选择去掉**。比反复调整更高效,且结果更对位内核。

### 推广

不止封面 Tag。任何创作元素都适用:
- 副标题 / 装饰文字 / 修饰元素 / 转场效果 / 滤镜 / 字幕特效 / ...
- "克制"的本质 = **敢于减法**
