## 一句话

把第01期"睡眠**方法**"的好反馈,顺着观众评论的真实痛点,深挖成第02期"睡眠**质量**",并用"痛点 ↔ 球星解法"映射,把大众痛点焊到明星热点上。

## 1. 选题缘起:从"方法"到"质量"

第01期讲达芬奇 vs C罗R90(睡眠方法),反馈好;观众评论暴露真实痛点:**睡得浅 / 半夜醒 / 睡不着**。第02期不另起炉灶,而是**沿同一热点(球星睡眠)向"质量"维度深挖**——同栏目复利,选题连续。

## 2. 建连接方法:痛点反向挂钩热点

不是"先有球星再找话题",而是**先收集观众痛点,反向去找球星/教练的对应解法**,做三栏映射:

| 观众痛点(带数据锚) | 球星解法 | 反转钩子 |
|---|---|---|
| 睡得浅(58.5%国人,睡眠白皮书) | C罗卧室=恢复舱(16-18℃/遮光/断屏) | "浅"常因环境太暖太亮,非体质 |
| 半夜醒(≥2次=质量下降) | R90:周期之间自然醒是设计 | 夜醒≠失眠,慌着盯钟才毁觉 |
| 入睡难(国人头号问题) | 锚"固定起床"+睡前90分钟降挡 | 别盯几点睡,盯几点起 |

→ 通用结构:**[大众痛点+数据] ↔ [明星同款解法] ↔ [反直觉反转]**。资料包见 `06_协作运营与发布/球星睡眠质量选题资料包_睡得浅夜醒入睡难.md`。

## 3. 钩子选择:挑最反直觉的痛点当主钩

三痛点里选**最反直觉、评论最易炸**的——"半夜醒≠失眠"做封面主钩,延续第01期验证过的"设问→反转"双层标题公式。

## 4. 视觉生产分工(定论)

- **封面底图 = MJ 水彩/水粉编辑插画**:高级、有质感。曾试 PIL 代码画底图 → 偏扁平卡通,味道不够,**弃用**。
- **信息/数据图 = PIL 代码精绘**:保证中文与数据准确(MJ 会出乱码假图)。
- 口诀:**情绪/封面交给 MJ,信息/数据交给代码**。

## 5. 构图踩坑 → 修正

封面初版:标题(左上)+ 人物(左下)同侧 → "偏沉"。修正:**水平镜像底图**,人物挪右下、标题留左上,形成对角平衡,且镜像后人物**脸朝标题**引导视线。已独立沉淀为 [[封面对角平衡构图律_v1]]。

## 6. 平台 / 合规硬约束

- **小红书笔记标题 ≤ 20 字**(发布框);封面**图片上的**标题不受限。本期标题从27字砍到16字「半夜总醒不是失眠?C罗教练这样说」。
- 不医疗化(不治疗/根治);"恢复/周期/身体年龄"类标注为教练/媒体说法,非医学结论。

## 7. 工作流闭环(可复用)

搜资料 → 痛点↔解法映射 → 定反直觉主钩 → MJ底图 + PIL信息图 → 构图 A/B/C 渲染复审 → 标题字数/合规 → commit & push → 复盘沉淀。
