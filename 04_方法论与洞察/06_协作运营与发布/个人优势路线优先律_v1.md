# 个人优势路线优先律（红海赛道：回到已验证的护城河，别在大众维度内卷）

> 入档：2026-06-29
> 来源：弄巧·七夕短片立项首日——先按"大众视觉 TVC"路线做，反复打磨视觉（红→含蓄、丝、簪、太暗）也只够 70-80 分；意识到后回到作者已验证的**音乐 MV 视听盛宴**优势路线（Stay Alive 17w 播放 / 1.3w+ 赞藏）。
> 验证状态：⚠️ 首次提出（策略层）；待弄巧成片数据二次验证。

---

## 一句话
红海赛道里，与其在**大众都在拼的维度**（如"视觉质量"）上内卷（天花板 70-80 分），不如回到**你自己已验证能起量的个人优势媒介/形式**做降维差异化。**投入前先定这条优势路线，别先烧几轮通用功夫才想起来。**

---

## 与「红海三路径」的层级区别
- [[红海主题的三条破局路径]] 破的是**题面**（一张图怎么不撞车）：概念 / 视觉奇观 / 媒介稀缺。
- 本律破的是**赛道 / 作品形态**（整个作品用什么形式交付）：把交付形态**服从于"你的护城河"**，而不是服从赛道的默认形态（赛道默认＝大众都在做的那种）。
- 衔接：本律可看作"路径三·媒介稀缺"的**个人化升级**——不是用"任意稀缺媒介"，而是用"**你已验证能起量的那个媒介**"。稀缺 + 已验证 = 真护城河。

## 核心三点
1. **别在大众维度内卷**：90% 的人都在拼"视觉高质量"时，你再拼也只是红海里的 70-80 分，难出彩。
2. **差异化来自"已验证的个人优势"**：不是任何不同，而是**你已有数据证明能起量**的那个能力（如音乐 MV）。别人抄不走你的个人风格 + 既往验证。
3. **优势路线前置**：接到题/赛道，**先问"我有什么别人没有、且已验证能起量的优势"**，让选题/风格/形态都服从它——而不是先按大众默认形态做，烧几轮才反应过来（弄巧首日的真实代价）。

## 操作
```
接红海赛道 / 红海题：
Step 1  先列"大众默认形态"（99% 的人会怎么交付）
Step 2  自问：我有哪个"别人没有 + 已验证能起量"的优势媒介/形式？
Step 3  有 → 让作品形态服从那个优势（降维差异化），别碰大众维度的内卷
        无 → 回退 [[红海主题的三条破局路径]] 走通用破局
```

## 边界
- 需要**真有**一条已验证（有数据）的个人优势；纯"我觉得我擅长"不算，否则变成自嗨。
- 优势媒介要能**接住当前题/赛道的硬要求**（弄巧仍需满足"非遗/手艺"的影子，只是用 MV 承载）。
- 没有已验证优势时，不适用本律——老实走红海三路径。
