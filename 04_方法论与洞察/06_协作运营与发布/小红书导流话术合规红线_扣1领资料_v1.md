# 小红书导流话术合规红线 ·「扣1领源码/教程」会被判广告限流

> 入档:2026-06-25
> 主线作品来源:`fraud-desk-demo/_promo/`(《反诈柜台》小红书发布包)
> 性质:小红书内容合规红线 + 发布物料自查规范

---

## 一句话

**「想要源码/思路,评论区扣『1』」这类关键词钓鱼式导流,会被小红书判定为引流广告 → 限流。配图卡、正文、置顶评论里都要清掉,改成"自然交流"引导。**

它和「建议收藏」是一对孪生陷阱:一个触发算法的"导流广告"判定,一个触发"保存型内容"降权。见 [[小红书点赞与收藏陷阱]]。

---

## 案例

《反诈柜台》发布包 `card4`(CTA 卡)原文案:

```
打开网站 · 直接玩
tiaozhuxiansheng.com/fraud-desk
国内直连不用翻 · 想要思路/源码？评论区扣「1」
```

正文同样写了「想要做法 / 思路:评论区扣『1』,我整理一篇保姆级教程」,发布建议里还让"置顶评论自己先发一条『想要教程扣1』"。

**三处同一套导流话术叠加** → 平台风控极易判为"诱导评论 + 站外引流",轻则限流、重则限制账号触达。

修改方向(已落地):

| 位置 | 改前 | 改后 |
|---|---|---|
| `card4.png` 配图 | 想要思路/源码？评论区扣「1」 | 浏览器打开即玩 |
| 正文 CTA | 想要做法/思路:评论区扣「1」…保姆级教程 | 做法和踩坑都写在评论区和后续笔记里,欢迎来聊 |
| 发布建议 | 置顶评论自己发「想要教程扣1」 | 置顶发「我是怎么一步步问 AI 做出来的」分享,引导自然讨论 |

---

## 为什么「扣1」会被打成广告

1. **关键词钓鱼**:用一个固定暗号(1 / 666 / 扣字)把用户行为标准化,这是营销号的典型特征,风控有现成模型识别。
2. **诱导评论**:平台不喜欢"为评论而评论"——它污染评论区质量信号(评论应是真实讨论,不是暗号刷屏)。
3. **隐含站外引流**:"领源码/领教程"潜台词是"加微信/进群/到站外拿",这正是小红书最敏感的红线。
4. **叠加放大**:配图+正文+置顶三处同时出现,等于反复向风控自证"我在导流"。

> 这条逻辑和小红书"重即时互动、压站外引流"的整体倾向一致——平台要把用户和交易都留在站内。

## 如何使用 · 发布前自查三条

### 1. 清掉一切"暗号 + 领取"结构

凡是「扣X / 回复X / 私我领 / 加我拿」配「资料 / 源码 / 教程 / 模板」的句式,一律删。哪怕你只是想做免费公益分享,句式本身就触发风控。

### 2. 把"导流"改成"已经给"

不要承诺"评论就给",而是把价值**直接放在内容里或后续笔记里**:

| 导流句(删) | 自然句(留) |
|---|---|
| 想要源码扣「1」 | 做法都写在评论区/下一篇笔记了 |
| 私我领教程 | 这个系列我会持续更,欢迎蹲 |
| 进群拿模板 | 踩过的坑都贴在评论里了 |

把互动钩子从"领东西"换成"聊观点/蹲更新",讨论是真实的,风控不打。

### 3. 外链只放自有站、且别配导流话术

`tiaozhuxiansheng.com/fraud-desk` 这种"自己的小站、国内直连、打开即玩"是相对安全的露出方式(产品本身,不是引流落地页)。但**别在它旁边再写"扣1领"**——一旦外链 + 暗号同框,等于把两条红线踩在一起。
外链本身的可访问性另见 [[小红书笔记外链必带xsec_token_v1]]。

## 反例:什么情况下可以引导互动

- **纯观点征集**:"你被骗过吗?评论区聊聊"——征集真实表达,不是发暗号领东西,安全。
- **真问题求助**:"这个 bug 有人遇到过吗"——真实讨论,鼓励。

界线:**引导的是"表达/讨论",还是"用暗号换站外资源"。** 前者平台要,后者平台杀。
