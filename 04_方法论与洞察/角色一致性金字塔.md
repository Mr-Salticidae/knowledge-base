# 角色一致性金字塔(4 层结构)

> 首次成型:2026-05-01(02 教程素材沉淀)

## 一句话总结

**角色一致性不是"三选一",是"四层金字塔"**——风格签名(Style Creator) / 角色身份(oref/seed/描述词) / 账号审美(personalize/moodboard) 各司其职。**叠用,不要选用**。

---

## 如何使用

写新作品前问自己:

1. **我要锁什么?**
   - 锁五官(同一个人)→ oref
   - 锁构图(同一帧的不同瞬间)→ seed
   - 锁装扮签名(IP 角色辨识符)→ 描述词
   - 锁作者签名风格 → Style Creator (sref)
   - 锁账号审美底盘 → personalize/moodboard

2. **我要锁几层?**
   - 长期 IP 系列 → 全部 4 层叠
   - 单条作品的多变体 → seed + sref
   - 不同场景下的同一个人 → personalize + oref + sref
   - 跨账号的我的审美 → personalize + sref

---

## 完整金字塔结构

```
顶层 · 风格签名(作者签名级)
└── Style Creator (sref)
    "让你画出'同一个作者笔下的人'"

第二层 · 角色身份(角色级)
├── oref       — 锁脸(她的生物身份)
├── seed       — 锁构图(她的戏剧瞬间)
└── 描述词锁定  — 锁装扮(她的 IP 标识)

第三层 · 账号审美(账号级)
├── personalize  — 锁审美方向(让 AI 学会"我属于谁")
└── moodboard    — 锁审美质感(告诉 AI"这就是我要的感觉")
```

---

## 每层的精度差异

| 层级 | 锁的精度 | 适用场景 |
|---|---|---|
| 顶层 · 风格签名 | "同一个作者笔下的人" | 跨平台 IP / 长期系列 |
| 五官精确(oref) | "同一个人" | 角色 IP / 真实人像替身 |
| 构图骨架(seed) | "同一帧的不同瞬间" | 系列变体 / 海报组 |
| 装扮签名(描述词) | "穿同款的那个调性的人" | IP 视觉标识 / 大类一致性 |
| 账号审美底盘 | "我的审美偏好覆盖所有题材" | 长期账号气质 |

---

## 类比 · 经典 IP 都靠"装扮签名"建立辨识度

- 路飞:草帽 + 红色背心 + 牛仔裤
- 哆啦 A 梦:蓝色 + 铃铛 + 红领巾
- 蜘蛛侠:红蓝战衣 + 蛛网纹

**读者认 IP 不是认五官,是认装扮签名**——五官随画风/视角/年龄都在变,**装扮才是恒定锚点**。

这正好对应"描述词锁定"的真实价值:你不能用文字精确描述五官(这是人脸生物特征的本质),但你**可以用文字精确描述"双侧白花 + 流苏耳坠 + 烟白汉服"**——这就是「檐下」的 IP 视觉签名。

## 顶层金句段(可直接用作正文压轴)

> *"一开始我以为三个方法是三选一——选最准的那个。*
> *实测后我发现:它们不是'三选一',是'三层叠加'。*
>
> *oref 锁住她的脸。*
> *seed 锁住她的构图。*
> *描述词锁住她的'装扮签名'。*
>
> *脸是她的'生物身份',构图是她的'戏剧瞬间',装扮是她的'IP 标识'。*
>
> *缺脸,她变成另一个人。*
> *缺装扮签名,她变成'穿汉服的某个东方少女'——和你的 IP 失之交臂。*
> *缺构图,你的图集没有海报感。*
>
> *真正的 AI 角色创作者从不把三个方法对立——*
> *他们用 oref 立人,用 seed 攒图集,用描述词建 IP。"*

## 升级:5/1 晚发现"风格签名"是新的顶层

原 3 层(oref / seed / 描述词)在 5/1 晚被升级为 4 层——
**Style Creator (sref) 是更高的"作者签名"层**。

类比:
- 角色 = 演员
- 风格签名 = 摄影师风格
- 账号审美 = 整个工作室的审美底色
