# 装扮签名 vs 五官精度 · IP 角色辨识的真相

> 首次沉淀:2026-05-01(02 教程方法 C 实测)
> 入档:2026-05-02
> 主线作品来源:`02_角色一致性锚点/08_方法C_反推prompts.md`

---

## 一句话总结

**真实 IP 角色都靠"装扮签名"建立辨识度,不靠"五官精度"**——这是描述词锁定法的真实价值,也是 IP 战略的隐藏真相。

---

## 如何使用

✅ **设计新 IP 时的优先级**:
1. 先设计 3-5 个**装扮签名**(头饰 / 服装 / 配饰 / 标志色)
2. 再设计五官特征(可塑性强,不必固定)
3. **永远不要**把 IP 的辨识度建立在"五官像谁"上

✅ **写描述词锁定段时的取舍**:
- 多写装扮(必锁,可被语言精确传递)
- 少写五官(锁不住,人脸是非语言的)

---

## 实测验证 · 方法 C 反推 4 张

跳蛛先生用「檐下」基准图反推得到"角色锁定段",拿去跑 4 个全新场景:

| 锁定的元素 | 4/4 锁住率 | 0/4 锁住率 |
|---|---|---|
| 双侧白花 | ✅ 100% | — |
| 流苏耳坠 | ✅ 100% | — |
| 烟白汉服 | ✅ 100% | — |
| 黑色长发 | ✅ 100% | — |
| 鹅蛋脸(粗) | ✅ 100% | — |
| **具体五官** | — | ❌ 0% |
| **唇色** | — | ❌ 0% |
| 整体气质 | 4 个略不同的"她" | — |

**结论**:**描述词锁了"装扮签名",没锁"五官精度"**。

## 类比 · 经典 IP 都是这样建立辨识度的

| IP | 装扮签名 | 五官 |
|---|---|---|
| 路飞 | 草帽 + 红色背心 + 牛仔裤 | 随作画风格变 |
| 哆啦 A 梦 | 蓝色 + 铃铛 + 红领巾 | 圆滚滚但具体五官无关 |
| 蜘蛛侠 | 红蓝战衣 + 蛛网纹 | 完全靠面具遮挡 |
| 米老鼠 | 大耳朵 + 红短裤 + 白手套 | 七十年来变了无数版 |
| 凯蒂猫 | 蝴蝶结 + 无嘴 | 极简形状 |

**读者认 IP 不是认五官,是认装扮签名**:
- 五官随画风、视角、年龄都在变
- 装扮才是恒定锚点

## 战略含义

### 1 · 「檐下」IP 视觉签名设计的正确性

「檐下」的视觉签名:**白玉兰 + 流苏耳坠 + 烟白汉服**
这 3 个元素:
- ✅ 都可以用文字精确描述
- ✅ 不依赖五官精度
- ✅ 在不同场景/姿势下都能保留

**所以「檐下」是个真正的 IP**,不是"那张图里的人"。

### 2 · 反向验证:「檐下」三分身的视觉签名差异化

按时间线 IP 战略,「檐下」三个分身:
- 18 岁:**白山茶 + 金流苏发钗**
- 22 岁:**白玉兰 + 流苏耳坠 + 水墨摄影感**
- 28 岁:**白玉兰 + 素色流苏耳坠 + 沉郁克制**

**三个装扮签名差异化设计**——确保读者一眼能区分,不依赖"五官变老了"这种弱信号。

### 3 · 写描述词锁定段时的取舍策略

写「檐下」描述词锁定段时:
- ✅ 多写:`single white magnolia tucked at hair crown`(装扮)
- ✅ 多写:`pale silk tassel earring partially hidden`(装扮)
- ✅ 多写:`off white Ming dynasty hanfu with stand collar`(装扮)
- ⚠️ 少写:具体五官(锁不住,浪费 prompt 字符)
- ⚠️ 少写:具体唇色(personalize 会覆盖)
