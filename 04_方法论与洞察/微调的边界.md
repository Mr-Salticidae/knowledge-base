# 微调的边界

> 入档：2026-05-16
> 主线作品来源：`03_prompt模板库/2026-05-16_闪电战双轮_R1+R2复盘.md`
> 性质：**工程规则** —— AI 图像迭代中"什么该改 / 什么不该改"的判断框架
> 关联：选错意象时调细节没用(待重新沉淀) / [[主体不看镜头律]]

---

## 一句话总结

**核心叙事意象不可微调**。微调前问：**"这个特征如果没了，主题还成立吗？"**——不成立就不动。

---

## 触发案例

R2「春风若有怜花意」获奖前的微调阶段，副会话提出了两个改动：

| 改动 | 评估 | 结果 |
|---|---|---|
| 单束光柱 → 整体柔光 | ❌ 错误 | 破坏了"春风的怜意"核心叙事 |
| 墓碑字迹清晰化 | ✓ 正确 | 增强细节，不破坏意象 |

**错误的那一个**：把"光柱 = 春风的怜意"这个核心叙事意象，当成"普通光线效果"处理掉了。**幸好跳蛛先生拒绝了**，否则获奖图就不存在了。

## 微调判断框架

### 可微调的（绿灯）

✅ **细节缺失**
- 缺一个手指、缺一个扣子
- 背景某个物体没画完整
- 字迹模糊（如墓碑碑文）

✅ **解剖错误**
- 五官比例失调
- 手部结构错误
- 多手 / 缺手

✅ **明显瑕疵**
- AI 拼接感
- 不自然的边缘
- 颜色错误

✅ **辅助元素**
- 配饰、衣服褶皱
- 远景小物体
- 装饰花纹

### 不可微调的（红灯）

❌ **构成画面情绪的不可替代意象**
- "墓碑"在 R2 里不是道具，是核心叙事意象
- 删了它整张图的题面解读就崩塌

❌ **构成对比 / 反差的核心元素**
- 灵光一现里的"灯泡头"不能改成"光晕"
- 拥挤通勤者里的"唯一灯泡头"不能改成"两个灯泡头"

❌ **驱动叙事的光线方向 / 色调**
- 单束光柱代表春风的怜意 ≠ 整体柔光（叙事崩塌）
- 冷暖对比代表内外世界割裂 ≠ 统一暖光

❌ **题面的核心隐喻**
- 任何能让"这张图为什么得奖"答案改变的元素

## 自检方法（重要）

**微调前**问自己 3 个问题：

```
□ 1. 这个特征如果没了，主题还成立吗？
       └─ 成立 → 可改
       └─ 不成立 → 不动

□ 2. 这个特征是不是评委 / 同行的"我没想到"触发点？
       └─ 是 → 不动（动了会失去触发器）
       └─ 不是 → 可改

□ 3. 这个特征是不是"反主流稀缺度"的来源？
       └─ 是 → 不动（动了会落回主流池）
       └─ 不是 → 可改
```

**任一问回答"不成立 / 是"** → **整体不动**。

## 副会话的常见错误（自我警示）

副会话在微调阶段最容易犯的错：

1. **把核心叙事意象当成"普通技术问题"**
   - 看到光柱 → 觉得"这个光不够柔和" → 想调成整体光
   - 没意识到这个光柱是叙事载体

2. **过度热心于"优化"**
   - 看到画面有"瑕疵感" → 想全部修掉
   - 没意识到瑕疵可能正是"我做不到"触发器（参考 [[我没想到+我做不到双触发器]]）

3. **基于自己审美 push 改动**
   - "我觉得这样更好看" ≠ "评委 / 同行会更投这个"
   - 副会话审美 ≠ 作品获胜逻辑

### 防护规则

副会话在跳蛛先生说"定稿" / "就这个" 之后，**不要再继续 push 任何改动**。

如果非要建议改动，必须先过"自检方法"3 问。**任一问亮红灯就停**。

## 推广 · 跨 AI 图像项目的适用性

这条规则不只适用于 Prompt Battle，**任何 AI 图像项目的迭代阶段都适用**：

| 项目类型 | 红灯（不可调）| 绿灯（可调）|
|---|---|---|
| Prompt Battle | 核心叙事意象 | 解剖 / 细节 / 辅助元素 |
| 商业插画 | 客户指定的核心元素 | 配色 / 排版 / 装饰 |
| 角色形象图 | 角色身份特征 | 背景 / 表情细节 |
| MV 卡点视觉 | 题眼镜头 | 过渡 / 微动 / 滤镜 |

**判断标准统一**：**"如果改了，作品的'存在理由'还在吗？"**
