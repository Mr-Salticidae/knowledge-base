# 封面对角平衡构图律

> 入档:2026-06-18
> 触发:45_AI拆解球星身体 第02期封面,跳蛛先生指出"偏沉——角色和文字都在左侧"
> 性质:图文封面/海报构图律,凡"底图有主体 + 需叠标题"的场景通用

---

## 一句话

**叠标题时,把文字放在底图主体的对角一侧**;若主体朝向背离标题,**水平镜像底图让主体脸/视线朝向标题**,借视线把读者引向文案。

---

## 问题现象(本次踩坑)

MJ 水彩底图:睡眠人物在**左下**,发光曲线对角升向右上暖点。我把标题也压在**左上**——

- 左上:深色标题(重)
- 左下:深色头发+衣物(重)
- 右下:整片留白(虚)

→ 文字与人物在左侧上下叠成一根"重柱",右下死空,视觉重量全沉左下 = **偏沉**。本质是**文字与主体没分开、挤在同侧**。

## 原理

1. **视觉重量平衡**:深色文字块与深色主体都是"重物",同侧堆叠 → 画面向该侧+底部倾倒。分置对角 → 两个重物互相牵制,画面稳。
2. **对角动线**:好底图本身常有对角动线(主体 → 留白 / 暖点)。标题落在主体对角侧,正好坐进动线终点的留白,顺势而非打架。
3. **视线引导**:人物脸/视线有方向性。让主体"望向"标题,读者视线被人物牵着走到文案上,提升标题被读到的概率。

## 操作清单

1. 先判断底图**主体在哪个角**(如左下)。
2. 标题放**对角**(右上/或保持上方但移到对侧)。本次:人物挪右下、标题留左上。
3. 主体背对标题时**水平镜像底图**(水彩/插画底图镜像无损;**信息图/含文字的图不可镜像**)。
4. **多版本渲染 A/B/C 对比定夺**,别凭脑补——本次跑了"原图标题居中 / 镜像标题左上 / 镜像标题居中"三版,镜像左上的对角平衡 + 脸朝标题最佳。
5. 标题落区做**局部奶油护字渐变**(只罩文字侧、不洗插画),保证可读又不毁底图质感。

## 边界 / 反例

- **主体居中或左右对称**的底图不适用,标题居中即可。
- **主体朝向有叙事意义**时(如人物望向窗外象征憧憬),不要为构图牺牲朝向。
- 镜像会翻转底图里的**文字/logo/惯用手**等,有这些元素时禁用镜像。
