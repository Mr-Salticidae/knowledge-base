# 时空叠化的锚定物律

> 入档:2026-05-15
> 主线作品来源:`03_prompt模板库/02_案例复盘/灵光一现+风花雪月双题复盘.md`
> 性质:多时态/多视角合成图的关键技巧

---

## 一句话总结

**当一张图要展示同一空间在不同时态(春夏秋冬、白天黑夜、过去现在)时,必须有跨象限的锚定物作为"这是同一空间"的物证**——没有锚定物,合成就退化为四张拼贴。

---

## 案例

「风花雪月」获奖图:画面被分为四个象限,每个象限是同一个东亚庭院在不同季节/元素下的状态(春樱、夏风、秋月、冬雪)。

**关键设计**:**板凳、圆窗、墙**这三个物体在四个象限里**位置一致**地出现。

prompt 里的核心句:
```
the bench wall and round window are continuous across the soft transitions 
creating one space layered with four poetic elements
```

没有这句话,MJ 会直接画四张独立的拼贴图;有了这句话,**四个象限被理解为"同一个空间在不同时刻"**。

---

## 为什么锚定物是必要的

人眼判断"这是不是同一个空间"靠的是**不变的标记物**:

- 没有锚定物 → 四个象限像是 PS 拼贴的四张图 → 主题退化为"四元素罗列"
- 有锚定物 → 四个象限是"同一个空间被时间穿过" → 主题升级为"时间的诗意"

锚定物本身不需要是叙事的中心,它只承担**"我证明这是同一处"**的元数据角色。

---

## 如何选锚定物

### 三个核心标准

1. **简洁** —— 形状简单、容易在不同光照/季节下保持可识别
2. **对称 / 几何感强** —— 板凳、窗框、柱子、桥、台阶,这类几何物比起复杂雕花更可靠
3. **跨季节可信** —— 物体本身应该是"四季都在"的(石头、木结构、金属)而不是季节性的(花、叶)

### 推荐锚定物类别

| 类别 | 示例 |
|---|---|
| 建筑结构 | 圆窗、门洞、屋檐、柱子、台阶 |
| 长期物件 | 木板凳、石碾、井、雕像 |
| 自然地貌 | 大石头、山轮廓、河岸线 |
| 几何符号 | 桥的弧线、墙的边缘、地砖纹理 |

### 不推荐做锚定物的

- 季节性物体(花、叶、雪堆)
- 易变形物(布、纸、火焰、水流)
- 人物 —— 人在不同时态会变化,作为锚定物太弱
- 动物 —— 同上

---

## 适用场景类型

本律不限于"四季合成",任何"多时态/多视角叠化"都适用:

| 场景类型 | 锚定物选择 |
|---|---|
| 四季叠化 | 建筑结构 + 长期家具 |
| 一天的时辰叠化 | 街道结构 + 固定建筑 |
| 过去 vs 现在(怀旧 vs 当下) | 老建筑 + 老物件(沿用至今的) |
| 现实 vs 梦境 | 主体身体 / 房间结构(不变,只换氛围) |
| 多视角(俯视+正视+侧视) | 同一物体的几何标志 |

## prompt 写法

### 必须显式声明锚定物的连续性

```
[the X and Y are continuous / aligned / consistent] across [the transitions / the quadrants / the panels],
creating one space layered with [different states / different times / different moods]
```

### 软化拼接的配套语句(避免硬切)

```
the transitions between [quadrants / panels] are gentle gradient blends not hard seams
```

→ 这条配合 [[反向描述的负面锚定]] 一起使用

### 统一色调避免色温割裂

```
unified soft palette of [color1] [color2] [color3], all in low saturation
```

→ 没这句话四个象限会色温飞机,失去"同一空间"的统一感

---

## 反例:何时不要叠化

- **主题本身就是"对比"或"撕裂"** —— 比如"善与恶"、"昼与夜的冲突",这种主题要的就是"两个不同的世界",硬合成会冲淡冲突
- **画面已经够复杂** —— 如果单一空间已经塞满道具,四时态叠化会糊成一片
- **核心情绪是"瞬间感"** —— 叠化是"时间长镜头",和"瞬间感"主题相反

## 二次验证与扩展（2026-06-12 · 对话甲骨文二进制获奖图）

**软化拼接条款二次验证 ✓✓**：「对话」获奖图 v2 因 `split-panel` / `dividing line` 被 MJ 字面执行成垂直硬缝；v3 用三件套修复成功：

```
one continuous illustration + within the same shared room   ← 空间声明
soft gradient color transition with no hard dividing line   ← 过渡显式化
--no hard split line, frame border                          ← 负面锚定
```

**适用边界扩展——过渡介质可替代锚定物**：本律原文要求锚定物证明"同一空间"。但当主题**就是**"两个世界的差异"时（对话/对比/古今类），两侧本来就该不同，锚定物反而无处安放——此时用一个**跨界过渡介质**（发光字符云/烟/光带）横跨两个半区即可：它既承担"软过渡"（接缝藏在介质后面），又承担"两个世界相连"的语义。

| 场景 | 用什么证明空间关系 |
|---|---|
| 同一空间多时态（四季/晨昏） | 锚定物（原条款）|
| 两个世界的对话/对比 | 过渡介质（扩展条款）|
