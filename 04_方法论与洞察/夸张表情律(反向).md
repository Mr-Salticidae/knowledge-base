# 夸张表情律(反向)

> 入档:2026-05-15
> 主线作品来源:`03_prompt模板库/灵光一现+风花雪月双题复盘.md`
> 性质:与「极简插画删表情律」配对,媒介切换时的方向反转

---

## 一句话总结

**漫画、油画、混合媒介这类强笔触语境下,夸张表情不是要"删"的冗余,而是核心引擎**——和写实摄影/极简插画的"删表情"完全反向。媒介决定方向,不是题材决定方向。

---

## 案例

「灵光一现」获奖图之一:男人脸极致特写,头顶半边是燃烧的灯泡,**瞪眼、张嘴**——表情戏剧化到极致。

这张图如果遵循 [[极简插画删表情律]],把表情删干净,就**完全做不出"被灵感击中的那一帧"的爆破感**。表情就是那张图的引擎。

---

## 为什么媒介决定方向

不同媒介对"表情"的承载能力完全不同:

### 写实摄影 + 极简插画(删表情语境)

- **写实摄影**:微表情已经在五官细节里了,显式写表情容易过度
- **极简插画**:笔触本身就没空间画微表情,显式写表情反而失去"留白"

### 漫画 + 油画 + 混合媒介(放大表情语境)

- **漫画**:粗线条 + 平涂阴影 = 笔触本身就在放大每一寸面部肌肉,不夸张就发挥不出媒介特性
- **油画厚涂**:厚涂笔触可以承载夸张的肌肉张力、瞪大的眼白、扭曲的嘴角
- **混合媒介**:多种语言冲突在同一帧,本来就需要"高密度信息"撑场,克制的表情会被淹没

---

## 决策树:写不写表情?

```
判断主体媒介:

写实摄影 →           不写明显表情,留给微表情
极简插画(MUJI 系) → 主动删表情,留白
漫画风 →             写夸张表情,放大五官
油画厚涂 →           写夸张表情,强调笔触承载
混合媒介(漫+油+网点)→ 极致夸张,媒介对冲表情
水彩 / 浮世绘 →      看具体流派,通常温和到中等
```

## 漫画/油画语境的表情 prompt 写法

### 强度阶梯

| 强度 | 关键词 |
|---|---|
| 中等 | `surprised` / `intense` / `focused` |
| 强 | `wide-eyed shock` / `mouth agape` / `dramatic stare` |
| 极致 | `eyes bulging` / `jaw dropped open` / `theatrical exaggerated reaction` / `over-the-top expression` |
| 漫画式标志性 | `manga-style exaggerated reaction` / `comic book style facial expression` / `every facial muscle visible` |

### 配套词

夸张表情往往配套以下放大器:

- **特写**:`extreme close-up cropping his head` / `tight framing`
- **切边**:`top of his head cut off by the frame` / `cropped off chin`
- **动态线条**:`speed lines radiating from his face` / `action lines emphasizing the moment`
- **网点纸**:`60s comic book halftone dots` / `manga screen tone`
- **粗线条**:`thick bold ink lines around the features`

## 与「极简插画删表情律」的关系

这两条律不是矛盾的,是**同一原则在不同媒介下的两种表达**:

**统一原则**:让媒介自己说话,不要让 prompt 和媒介打架。

| 媒介承载力 | 表情策略 |
|---|---|
| 媒介无力承载表情(极简、写实) | 删表情,让媒介的"细微"或"留白"发挥 |
| 媒介擅长承载表情(漫画、油画) | 放大表情,让媒介的"粗笔触"发挥 |

**核心问错**:不是问"主体什么情绪",而是问"这个媒介能演什么".

---

## 反例:何时夸张表情会翻车

- **写实摄影硬写夸张表情** → 立刻变成"演员摆拍"或"AI 脸"
- **极简插画硬写夸张表情** → 五官变细了,画面变成廉价漫画感
- **MUJI 风 / 长场雄风** → 这些风格的核心就是"克制",夸张表情会破坏品牌感
