# 网易云音乐人发布 SOP · v1

> 入档：2026-05-15
> 触发：「跳蛛先生的不正经实验室」专辑上传 · 新宿瓜摊夜曲首发
> 性质：操作手册 + 字段填写决策框架
> 关联：[[好流量是好作品的产物_v3.1反思]] / [[快手分发SOP_v1]]

---

## 一句话

**网易云不是流量平台，是音乐资产的"长期家"——视频平台生命周期 7-30 天，网易云是数年**。所有字段填写决策围绕"让这首歌在 3 年后被检索到"做，不是"让算法明天推荐"。

---

## 字段全清单

网易云音乐人发布每首歌需要填以下字段：

| 字段 | 必填/选填 | 上限 |
|---|---|---|
| 曲风标签 | 必填 | 最多 2 个 |
| 情绪标签 | 选填 | 最多 3 个 |
| 场景标签 | 选填 | 最多 3 个 |
| 主创说 | 选填 | 文字+语音，评论区顶部展示 |
| 私信通知粉丝 | 选填 | 推送给现有粉丝 |
| 发布站内笔记 | 选填 | 网易云站内社交分发 |
| 封面 | 必填 | 1:1 比例，建议 1080×1080 或更高 |
| AI 创作标注 | 必填（如适用）| 勾选/不勾选 |

## 字段填写决策框架

### 曲风标签（必填，最多 2 个）—— 决定算法分发圈

**网易云的曲风标签是算法分发的核心信号**。选错会让歌被推给完全错误的人。

可选项：流行 / 嘻哈说唱 / 电子 / 中国传统 / 轻音乐 / 慢摇DJ / 民谣 / 摇滚 / 国风 / 另类独立 / 实验 / 民族歌曲 / 原声带 / 世界音乐 / 二次元 / R&B / 戏曲 / 古典 / 金属 / 新世纪 / 儿童 / 爵士 / 蓝调 / 乡村 / 雷鬼 / 拉丁音乐 / 其他 / 放克

**决策原则**：

1. **主标签** = 选最接近的"音乐流派"——这是算法识别这首歌该被推给谁的依据
2. **副标签** = 选"扩展受众"——把听主标签的人池子扩大
3. **不要为了"高级"选「另类/独立」**——除非作品真的另类独立，否则算法会把你推给小众独立圈，而你的真正受众在主流圈

**避坑**：
- ❌ 不要选「实验」作为主标签——「实验」标签的浏览量极低，等于沉底
- ❌ 不要选「其他」——算法会忽略
- ❌ 不要选「二次元」除非作品真的是 ACG 向——会被分发到错误受众

### 情绪标签（选填，最多 3 个）—— 决定推荐位匹配

可选项：狂欢 / 热血励志 / 活力 / 欢快 / 甜蜜 / 自信 / 浪漫 / 放松 / 治愈 / 思念 / 平静 / 孤独 / 悲伤 / 苦情 / 抑郁

**决策原则**：

1. 选作品"听完后留下的情绪"，不是"创作时的情绪"
2. **3 个标签应该是同一情绪光谱的不同浓度**，不是 3 种对立情绪——例如"孤独+思念+平静"成立，但"孤独+欢快+狂欢"会让算法判断不准
3. 默认情绪要"准"而不要"亮"——选"平静"比选"治愈"更安全

**避坑**：
- ❌ 不要选「热血励志」/「狂欢」除非真的是——这两个标签会被推到完全错误的鸡汤/广场舞类受众池

### 场景标签（选填，最多 3 个）—— 决定使用场景匹配

可选项：学习 / 约会 / 夜晚 / 工作 / KTV / 助眠 / 小酒馆 / 运动 / 驾车 / 游戏直播 / 清晨

**决策原则**：

1. 选作品"用户最可能听这首歌的场景"
2. 3 个场景应该有真实关联（夜晚+小酒馆+驾车 = 都属于夜间私人时间），不要凑数
3. 「KTV」和「游戏直播」是流量较大的标签，但选错了反而被埋

**避坑**：
- ❌ 「KTV」标签只适合可翻唱的流行歌——AI 自动作词的歌没法被人翻唱，不要选
- ❌ 「游戏直播」标签的受众期待无人声/低人声的 BGM 类——人声主导歌曲不选

### 主创说（选填，评论区顶部展示）—— 直接面向听众

**决策原则**：

- **不解释音乐技术细节**（"用 Suno 做的""prompt 是这样的"）——这破坏作品的"完整感"
- **维护作品的世界观语境**——如果歌曲假托某个虚构背景（如西瓜的"1985 City Pop Records 限定盘"），主创说继续维护这个语境
- **不要写"这是我的第 X 首作品"**——这是创作者视角，不是听众视角

**好的主创说应该回答**：这首歌是从哪个世界传来的？

### 私信通知粉丝—— 推送给现有粉丝

**决策原则**：

- 比主创说稍微"出戏"一些，可以说创作者视角的事（如"开了新专辑"）
- 不要太长——粉丝在私信里看不会读长文
- 引导一个具体动作（听 / 评论 / 收藏）

### 发布站内笔记—— 网易云站内社交分发

**决策原则**：

- 比私信再正式一档——这是发到公开圈层的内容
- 加 #话题# 标签——网易云的站内话题流量是真实的
- 可以提工具栈（"AI 二创")——网易云的"AI 创作"圈层是有的，不像视频平台需要弱化
- 不要写过度自夸的话——网易云用户对"求关注"敏感

### 封面（必填，1:1）

**决策原则**：

- 必须 1:1 方形（1080×1080 起步）
- **不要用视频封面的截图作为唱片封面**——它们是两种媒介
- 唱片封面的视觉传统是"标题居中 + 主体居下"
- 推荐复用专辑系列封面的视觉语言——形成专辑识别度

### AI 创作标注

**决策原则**：

- 网易云规则要求标注，**必须勾选**
- 不勾选有被下架风险
- 勾选不会显著影响曝光——网易云"AI 创作"标签有独立流量池

## 关于专辑架构 · Vol.X 分卷模式（2026-05-15 拍板）

### 背景

网易云的"单曲不能单独有封面，只能用专辑封面"规则，迫使决策"专辑 IP 怎么组织"。

考虑过三个方案：
- A. 每首歌独立成单曲专辑（专辑名 = 歌名）→ ❌ IP 不可见
- B. 一张合集专辑持续填充 → ⚠️ 需新做 IP 标识封面 + 调性约束
- C. **Vol.X 分卷专辑**（每张 Vol 一张专辑，封面随该卷主歌定）→ ✅ 推荐并拍板

### Vol.X 分卷模式的操作规则

1. **专辑命名规则**：「[IP 短名] Vol.N」——例如「不正经实验室 Vol.1」
   - IP 短名建议精简（艺人名"跳蛛先生"由音乐人账号字段自动带出，不重复）
   - 不要写"vol.1"用"Vol.1"（首字母大写，符合英文唱片惯例）
2. **每张 Vol 收录歌曲**：原则上 1 首（致敬黑胶 Single 传统）/ 上限 3 首（如果是有强主题关联的小合集）
3. **封面随该 Vol 主歌**——不需要做"IP 标识封面"
4. **致敬黑胶 Album Series 传统** —— David Bowie / Pink Floyd 等历史先例

### Vol 命名一致性约束

未来发布新 Vol 时必须保持：
- 「[IP 短名] Vol.N」格式不变（N 是自然数序列：1 / 2 / 3 ...）
- 不跳号、不分支（不出现"Vol.1.5"或"Vol.2 重制版"）
- 失败的实验作品**不发**，不用 Vol.N 占编号

## 实际案例 · 不正经实验室 Vol.1 · 新宿瓜摊夜曲

### 专辑定位

**跳蛛先生的不正经实验室**——专门收录"把世界上最不正经的东西做得最正经"的实验作品。每首都是一次创作实验：把一段全民共识扔进 AI 工具链里，看它会变成什么。

### Vol.1 · 第一首 · 新宿瓜摊夜曲

**曲风标签**：
- 主标签：**电子**（City Pop 在网易云分类下最接近"电子"）
- 副标签：**另类/独立**（捕捉"实验性 + 文化混搭"调性）

**情绪标签**（3 个）：
- **孤独**（深夜瓜摊主的内核）
- **思念**（80 年代怀旧感）
- **平静**（City Pop 的舒缓底色）

**场景标签**（3 个）：
- **夜晚**（绝对契合）
- **小酒馆**（City Pop 的天然听场）
- **驾车**（深夜城市开车的背景音）

**封面**：`18_这瓜包熟吗/cover_1_1_网易云封面_最终.png`（从 cover_4_3 偏左裁切得到，完整保留标题/人物/瓜山/霓虹/SIDE A 标记）

**AI 创作标注**：✅ 必勾

**主创说**（评论区顶部）：
```
1985 年昭和 60 年，新宿歌舞伎町。
霓虹在湿润的柏油路上化开。
一位旅日华人摊主站在他的瓜山前，
唱出了那个年代最迷惘的关于「保熟」的疑问。

收录于 City Pop Records 1985 限定盘。
原盘已绝版，本歌为 AI 数字修复版本。
```

**私信通知粉丝**：
```
开了新专辑《跳蛛先生的不正经实验室》。
专门收录把不正经的东西做得最正经的实验作品。
第一首：新宿瓜摊夜曲。
你笑没笑？告诉我。
```

**发布站内笔记**：
```
#AI音乐 #CityPop #实验作品 #刘华强

新专辑「跳蛛先生的不正经实验室」上线。
专门收录那些「把世界上最不正经的东西做得最正经」的实验作品。

Vol.1 · 新宿瓜摊夜曲
假托 1985 年昭和 60 年 City Pop Records 限定盘 SIDE A，
AI 数字修复版本。
所有人声 / 旋律 / 编曲均由 AI 完成。

如果你笑了，告诉我笑在哪一句。
```

## 后续 vol.2+ 的字段一致性原则

未来这个专辑的其他单曲发布时，**保持以下变量一致**：

1. **专辑名固定**：跳蛛先生的不正经实验室
2. **封面视觉语言固定**：标题居顶 + 主体居中 + 底部 SIDE A 标记 + 复古唱片质感
3. **主创说统一调式**：维护各歌曲的"虚构世界观"语境，不解释 AI 流程
4. **AI 创作标注一律必勾**

**允许变化的**：曲风标签、情绪标签、场景标签（按具体作品调整）
