# 情绪散文当 prompt 主体 · v1

> 入档:2026-06-30
> 来源:MJ 官网发现的 prompt + sref 组合包(详见 [[sref_版画情绪散文四签名_3565168336]])
> 一句话律:**把 prompt 的主体从「描述一个画面」换成「写一段情绪独白」,MJ 会自动从中抽名词当母题、借语气当滤镜,一次出一组同款不同画的发散图。**

---

## 反直觉点

常规 prompt 教程都在教「写清楚画面」——主体、动作、构图、光线越具体越好(参 [[prompt的三段式结构_v1]] / [[prompt极简化原则_v1]])。

这套玩法**反着来**:主体位置塞的不是场景,而是一整段第一人称散文,只谈感受和日常名词,完全不写构图。例:

> *"I don't know how to talk about flowers. They're so colorful... I don't know how to talk about a bunch of other things, like breakfast... What should I say about cities... life, love, reality, imagination, always speak for themselves."*

MJ 读不懂这段话的哲学含义,但它会做两件事——

1. **抽名词当母题**:flowers / sun / water / breakfast / cities / person → 渲染成具体物件(花、咖啡、吐司、煎蛋、脸)。
2. **借语气当滤镜**:忧郁、温柔、不确定的口吻,把输出整体压向安静怅惘(蔫花、低垂的脸、流泪的眼、骷髅=必死)。情绪不被直接画出,而是给画面**上色**。

因为散文**模糊、不规定**,MJ 自由度极大 → 同一条 prompt 出的一组图差异很大,每张是同一情绪的不同联想。**「读不清」在这里不是 bug,是发散性的来源。**

---

## 分工心法(关键)

一条这样的 prompt,三个部件各管一摊,别混:

| 部件 | 管什么 | 例 |
|---|---|---|
| **风格词 + sref** | 长什么样(视觉签名) | 双色版画、木刻质感(见 [[sref_版画情绪散文四签名_3565168336]]) |
| **情绪散文** | 画什么 + 什么情绪(母题 + 滤镜) | 花 / 早餐 / 脸,忧郁怅惘 |
| **媒介前缀** | 哪种媒介 | `hand drawn illustration of` |

**口诀:sref 管脸,散文管心,风格词管皮。** 三者正交,换其一不动其二。

---

## 什么时候用 / 什么时候别用

✅ **用**:要一组**情绪统一、画面发散**的图——招贴集、zine、专辑内页、情绪海报墙、系列封面。拥抱变化,挑选而非控制。

❌ **别用**:要一张**指定构图**的图(主体在哪、看哪、几个人)。散文等于把控制权让出去了,这时老老实实描述画面。

> 它和 [[prompt极简化原则_v1]] 是**一对反向工具**:极简化是为了**收敛、可控**;情绪散文是为了**发散、捞惊喜**。同一个坐标轴的两端,按需取用,别混。

## 可复用模板

```text
[媒介前缀,如 vintage poster illustration / hand drawn illustration of]
[一段第一人称情绪独白:只谈感受 + 日常名词,不写构图、不写动作、不写镜头]
, thin outline, minimal shading, clipart, vintage poster
--sref A B C D
```

写散文那段的三条纪律:
1. **第一人称、谈不确定**("I don't know how to talk about...")最容易出怅惘味;换成笃定/欢快口吻,滤镜也跟着变。
2. **多撒具体名词**(花、早餐、城市、信),它们就是母题种子;名词越具体越好画。
3. **别写构图词**(left/center/close-up/looking at),一写就把发散性压没了。

---

## 举一反三

- 换情绪轴:把忧郁独白换成**童年回忆 / 思乡 / 暗恋**的第一人称碎碎念 → 同套 sref 出另一组母题。
- 换签名:散文不动,sref 换成水墨 / OVA / MUJI 签名 → 同一段心事换一种皮。
- 做系列:固定 sref + 风格词当「栏目视觉」,每期换一段散文当「这期心事」,天然成系列。
