# 风格基底前置决策 v1

> 入档：2026-06-12
> 来源：[[2026-06-12_对话甲骨文二进制获奖图复盘]] 下次改进条目（v1 摄影版弯路的反推）
> 验证状态：⚠️ 首次发现（1 次反面教训），暂时性结论

---

## 一句话总结

**风格基底是 prompt 的第一个分叉口——动笔写 prompt 前，先回答"摄影感还是插画感"，再开始写。**

---

## 来源教训

「对话」获奖图的 v1 摄影版：概念（甲骨文 × 二进制对刻）已确认成立，但默认套用了摄影词系（cinematic / volumetric light / chiaroscuro），出图审美不达标，被迫整轮置换风格基底重跑——**这一轮本可以不存在**。

弯路的根源：把风格当作"修补项"（不好看再调），而它实际是**最上游的决策**。概念层之下的所有词（光影、镜头、材质、负面词）都依附于风格基底，基底错了整组词全要换。

---

## 操作规则

1. **概念确定后、动笔前，先问一句**：这张图要摄影感还是插画感？（拿不准就问需求方/看参考图）
2. 风格基底决定后，**整组词系才能确定**：
   - 摄影系 → cinematic / color grading / 镜头语言 / 景深 / `--no illustration, painting`
   - 插画系 → flat / linework / editorial / 色块 / `--no photorealistic, 3d render`
3. 有参考图时，**风格判断以参考图为准**，并直接移植其词汇（见 [[概念锚定_风格置换迭代法_v1]] 第 3 条）
4. 决策依据可参考：投放场景（海报/文章配图/社交媒体）、主题温度（纪实感选摄影、概念感选插画）、比赛池的差异化空间
