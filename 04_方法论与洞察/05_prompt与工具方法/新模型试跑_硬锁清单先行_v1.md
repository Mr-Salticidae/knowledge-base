# 新模型试跑 · 硬锁清单先行律 · v1

> 入档：2026-07-03
> 触发：49_凝视_写实重置 EP02，MJ v8.2 老 prompt 直跑踩坑
> 关联：[[MJ_v8.2_行为档案_v1]] / [[角色一致性金字塔]]

---

## 一句话律

**新模型的质感兴奋会掩盖设定违规——直跑试水可以，入库前必须逐条过角色卡硬锁清单。**

---

## 踩坑现场

MJ v8.2 上线，拿 VESPER 老 prompt 直跑试质感：5 张全部"好看到想直接用"。但角色卡硬锁是**异色瞳（冷蓝+暖粉红）必须、明令禁止统一瞳色**——5 张全部双眼统一 violet-blue，全军违规。

坑的机制：**注意力被新模型的质感提升全部吸走**，"哇好看"压过了"等等，这对不对"。如果不是入库归档时对了一遍角色卡，这批违规图就直接当定妆主图用了——异色瞳恰恰是该角色"看见两面"设定的视觉化核心，丢了等于 IP 识别塌一半。

---

## 操作规则

1. 新模型试水允许老 prompt 直跑（快速验证质感值不值得换版本）
2. **入库时逐条对硬锁表打勾**——角色卡"必须"项一项项核对，不凭整体印象
3. 违规图不作废，**降级**为"质感验证 / 氛围候选 / 虚焦过场素材"，不得当定妆主图
4. 修复走**最小变量补跑**：只改违规项的短语 + sref 原批锁质感，别整段重写（本次仅换瞳色短语即 6/6 修复）

---

## 边界

- 无角色卡的一次性出图 / 纯探索发散不适用
- "软偏差"（如发型长短这类"可调整"项）不走本律，但要**当场裁决并回写角色卡**，不许事实默认
