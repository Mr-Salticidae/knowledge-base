# 冷门风格词的压缩律 · v1

> 入档:2026-06-30
> 来源:作者在 MJ 上发现的风格词 `Monnem design`,用两条极简 prompt(男/女各一条)各出 8 图,风格高度统一且质量很高。
> 验证状态:`Monnem design` 这一个词的承载力 = ✅ 已复现(2 prompt × 8 图 × 2 性别);**「冷门风格词普遍能承载复合风格」这条通则 = ⚠️ 首次发现(单词验证),待换词二次验证。**
> 一句话律:**一个冷门专有名词当风格词,可以独自压缩进「版面 + 印刷质感 + 配色 + 情绪 + 人像气质」一整套复合美学;此时 prompt 只需 `风格词 + 主体一句话 + 景别`,把控制权交给这个词,质量比你自己堆形容词更高更稳。**

---

## 反直觉点

常规教学:想要高质量,就把画面写细——版式、配色、质感、光线、情绪逐项描述(参 [[prompt的三段式结构_v1]])。

这次反着来也成立,而且更省力:**找到一个对的「冷门风格词」,一个词就替你把五六个维度全锁了。**

作者实测两条 prompt,没有任何 sref、没有任何风格形容词堆砌:

```text
Monnem design, a cool asian young guy's poster, upper body
Monnem design, a cool asian young woman's poster, real photo, upper body
```

16 张出图全部落在同一套高完成度的「平面海报 + 模拟印刷」美学里。也就是说 `Monnem design` 这一个词,**单独承载了下面整套东西**:

| 被一个词锁住的维度 | 这个词带来的具体表现 |
|---|---|
| **版面 / 构图** | 海报式居中主体、留白、几何框、圆形色块(头后一轮"日/月")、撕边/做旧边框 |
| **印刷质感** | 丝网/Riso 版画感、粗颗粒、半调网点、套印错位味 |
| **配色** | 双色调(teal+橙 / teal+粉 / 单色蓝)、大块平涂色条压边 |
| **情绪滤镜** | 忧郁、疏离、冷淡,东亚青年特有的"emo 编辑感" |
| **人物气质** | 不看镜头 / 低垂 / 侧脸,高领毛衣·西装·吊带,墨镜 |

> 你自己要写出这套效果,至少得堆 `risograph print, duotone, halftone grain, screenprint texture, editorial poster layout, geometric color blocks, melancholic mood, muted palette...` 一长串,还未必稳。**一个对的冷门词把这串全压缩了——这就是它的杠杆。**

为什么是"冷门"词更好用:大众词(`cinematic`、`portrait photography`)被海量图喂过,语义被均摊、风格被拉平 → 出图泛、要靠堆词才出彩;冷门专有名词(地名、小众设计圈黑话、工作室名)在训练集里**只对应一小撮高度同质的高质量图**,于是这个词的"风格指纹"又窄又利,一调就准。(`Monnem` 据查无公开释义/出处,搜不到反而印证了它的小众——但能不能用**不取决于你知不知道它什么意思,只取决于出图稳不稳**。)

## 这是哪一类工具(坐标定位)

这条律和库里另外两条是**同一根坐标轴**上的配套件,别混用:

| 工具 | 风格词在 prompt 里的角色 | 干嘛用 |
|---|---|---|
| **本律:风格词当引擎** | 风格词 = 几乎整条 prompt 的发动机,主体只补一句 | 用一个词换一整套高完成度美学,极省力 |
| [[情绪散文当prompt主体_v1]] | 风格词只"管皮",散文"管心"(母题+滤镜) | 要一组情绪统一、画面发散的图 |
| [[prompt极简化原则_v1]] | 风格词 + 极简主体,收敛可控 | 要一张构图明确、可复现的图 |

口诀对照:情绪散文那篇说「**sref 管脸、散文管心、风格词管皮**」;本律说的是当**「皮」这一层本身就足够强、足够复合**时,可以让它直接当主角,脸和心都退到一句话带过。

## 方法步骤(怎么找 + 怎么用一个载重风格词)

1. **找候选词**:留意冷门专有名词——地名/方言名(`Monnem`=曼海姆当地叫法一类)、小众设计运动、工作室名、杂志名、某摄影/版画流派黑话。来源:别人的 prompt、MJ 官网/社区、设计圈术语。
2. **裸测**:`风格词, a portrait, upper body` 这种近乎空的 prompt 直接出 4 图,**别加任何风格形容词**——目的是看这个词"素颜"能扛多少。
3. **判载重**:看这一组 4 图是否**自动统一**地带出了多个维度(版式+质感+配色+情绪)。
   - 一个词就锁住 ≥3 个维度且质量高 → **载重词,留下当引擎**。
   - 只影响单一维度(只是变个色调/只换个媒介)→ 普通修饰词,回去老老实实堆 prompt。
4. **最小化骨架**:确认载重后,prompt 收到极简:
   ```text
   [载重风格词], [主体一句话], [景别]
   ```
   主体只给"谁 + 大致气质"(`a cool asian young guy`),景别给一个(`upper body`)。**别再加风格词**——加了反而和载重词打架、把它的指纹冲淡。
5. **要变体就换"主体一句话"或加一个开关词**:如本案 `real photo` 把女版从平面海报推向更摄影的颗粒感;`guy/woman`、`real photo`、`upper body/full body` 都是低风险开关,风格底盘不动。
6. **拥抱发散、批量挑**:一条词出一组,差异主要在配色/版式排布——这正是它的好处,当系列封面/招贴墙用,挑而非控。

## 可复用模板

```text
[载重风格词], a [气质形容词] [人物] poster, [可选 real photo], upper body --ar 3:4
```

本案两条已验证底料:

```text
Monnem design, a cool asian young guy's poster, upper body
Monnem design, a cool asian young woman's poster, real photo, upper body
```

三条纪律:
1. **主体只写一句**,把版面/质感/配色全留给风格词,别替它操心。
2. **不堆风格形容词**:载重词已经把皮锁死,你加的词只会内耗。
3. **变体走"开关词"**(real photo / 性别 / 景别),不走"换风格词",否则底盘就变了。

## 适用边界 / 排错

- ✅ **用**:要海报/封面/招贴/专辑视觉,要"一套统一的高级皮",且**不强求指定构图**(主体看哪、几个人、手放哪都随它)。
- ❌ **别用**:要**精确构图/指定动作/角色一致性**——载重风格词的发散性会把这些冲掉,该回 [[prompt极简化原则_v1]] 或上 sref/oref。
- **排错·出图飘了**:十有八九是你又加了风格形容词或 `--style raw` 之类把词的指纹冲淡了 → 删干净,只留载重词。
- **排错·风格不够强**:这个词可能根本不载重(步骤 3 没过关),换词,别试图用形容词把它"补"起来。
- **风险·不可控/不稳定**:冷门词在不同 MJ 版本里的指纹可能漂移(版本更新会重训);**当下能复现 ≠ 永久稳定**,重要项目要存好出图、记下版本号。

---

## 举一反三

- **建一张"载重词清单"**:每发现一个过了步骤 3 的词,记下它锁住了哪几个维度、最佳裸骨架,攒成自己的私有风格库。
- **载重词 × 情绪散文**:把本律的"主体一句话"换成 [[情绪散文当prompt主体_v1]] 的一段独白 → 用载重词当栏目视觉、用散文当每期母题,天然成系列。
- **载重词 × sref**:载重词管皮 + sref 管脸,做"同一套海报美学下的固定角色"(参 [[sref_版画情绪散文四签名_3565168336]] 的版画情绪人像也是同一审美家族,只是它走 sref 路线、本律走单词路线——可对照取用)。
