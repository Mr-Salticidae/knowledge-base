## 一句话

杂志海报里「一张照片被切成断裂网格、缝隙露底」的高级感，**不是拼九张缩略图**——是**整张图铺满网格区、每格只显示它盖住的那一块**。认清这一点，用 Pillow 二十行就能复刻，且能套任何竖构图英雄图。

---

## 关键机制：连续图 → 断裂网格（本文核心）

参考模板的视觉是：一张连续照片被切进 3 列断裂网格，格与格之间留缝、缝里是底色，眼睛会把碎片**读成一整张**。新手最容易走错的实现是「把图裁成 9 块分别贴」——那样缝隙两侧对不齐，立刻露馅。

正确做法**只有一步认知差**：

```python
# 1) 整张原图按 cover 缩放，盖满「整个网格区域」(gw×gh)，得到一张连续大图 grid_img
grid_img = cover(src, gw, gh, bias_y=0.28)   # bias_y<0.5 偏上裁，留住人物头部
# 2) 每个格子 tile(tx,ty,tw,th) 只从 grid_img 上「对应坐标」抠出自己那块贴回画布
crop = grid_img.crop((tx-gx, ty-gy, tx-gx+tw, ty-gy+th))
canvas.paste(crop, (tx, ty))
# 缝隙不贴任何东西 → 保持底色。于是碎片自动对齐成一整张。
```

一句话记法：**先把整图铺满网格区，再让每格"开窗"看自己那块**。缝隙 = 底色 = 断裂感来源。

副作用是好事：人物落在中间列被切开、两侧格子露出拼贴/半调背景——和参考模板"主体落一格、其余格露空草地"是同一个构图逻辑，天然高级。

## 断裂网格的排布（仿参考模板的破格律动）

3 列 + 列间缝，纵向分 4 个 band（前三矮、末一高），band 内做「宽格 + 窄格」错位，就得到参考模板那种不呆板的破格：

| band | 高度占比 | 格子 |
|---|---|---|
| 1 | 0.19 | 三等格 |
| 2 | 0.19 | 宽格(占 2 列) + 窄格 |
| 3 | 0.19 | 宽格(占 2 列) + 窄格 |
| 4 | 0.43 | 三个高格（主体落这里最抓眼） |

列宽 `colW=(gw-2*缝)/3`，宽格 `=2*colW+缝`。改这张表就能换律动，不必动机制。

## 配色：底色不抄模板，从源图里采

参考模板是绿调，因为它底图是绿草地——**底色跟着源图走，海报才整体**。这次源图是脏灰+霓虹绿/品红的朋克拼贴，就把底色做成**脏橄榄墨绿渐变**（脱胎自参考模板的绿、拉向源图灰调），点缀色直接取源图的**荧光绿 + 品红**（标题下短横、角落 N° 小字）。

判据一句话：**底色 = 源图大面积色的降饱和暗化版；点缀 = 源图最跳的那一两个色**。亮调题材（如小白编辑风）就反过来做奶粉底 + 深墨字——见 [[代码排版小红书封面工作流_v1]] 的"封标随主题切设计语言"。

## 完整流程（Pillow）

```
底色渐变 (对角, 源图采样降饱和) + 柔暗角
 └─ 标题 (Arial Bold) + 品红短横 rule
     └─ 正文块 (4 行, 浅灰)
         └─ 断裂网格：整图 cover 铺满网格区 → 逐格开窗贴切片 → 缝留底色
         │    └─ 每格 1px 深色描边 + 整组底部柔影(GaussianBlur)提层次
         └─ 左下署名 (create/日期/by) + 右上角 N° 霓虹小字
             └─ 右下虹彩玻璃装饰 (两根交叉棱柱, 竖向虹彩渐变 + 白高光棱线 + 柔光晕)
```

要点：
- **偏上裁** `bias_y=0.28`：竖构图人物头在上部，cover 到偏方的网格区会上下裁，偏上裁保住脸。
- **字体优雅降级**：`font()` 按候选路径找，找不到退 `load_default()`，换机器/Mac 不崩。
- **玻璃装饰**是代码画的：一根棱柱 = 拉长六边形，沿长轴铺 60 条竖向渐变线（虹彩色表插值）+ 中缝白高光 + 外描白边，两根交叉即成"水晶十字"。不追求照片级，够呼应模板的装饰位即可。

## 复用与排错

- 换作品：拷 `make_poster.py` 改顶部 `SRC/OUT/TITLE/BODY/CRED` 与配色即可。
- **源图选竖构图**（3:4 最佳）：主体贯穿全高，被网格切开最好看；横图会被裁掉头尾。
- 装饰太乱 → 减棱柱数量（两根足够），别叠三根以上。
- 缝隙对不齐 → 你多半退回了"分块裁图"老路；回到"整图铺满 + 逐格开窗"。
- 找不到原图别翻 temp → 用缩略接触表视觉检索，见 [[代码排版小红书封面工作流_v1]] 卡点 1。

## 举一反三

这套「连续底 + 断裂开窗」不止做人物海报：
- 产品图切网格 → 电商 banner；
- 一张风景切网格 + 大留白标题 → 旅行/展览海报；
- 甚至把"底"换成视频每帧，格子固定 → 断裂网格转场（静态装饰预烤成 PNG，参考 [[代码排版小红书封面工作流_v1]] 的"静态元素预烤"提速律）。

判断能不能用：**你想要"一整张被切开"的高级感、且主体是单张竖图** → 用它。
