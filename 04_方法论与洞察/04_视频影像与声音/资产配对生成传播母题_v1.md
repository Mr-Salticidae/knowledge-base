# 资产配对生成传播母题

> 入档：2026-06-16
> 来源：《Skill Is All You Need》v3.1 加宣传噱头环节。
> 性质：从已有资产生成传播母题（宣传噱头）的方法。

## 核心

给作品找"宣传噱头 / 母题"时，不要凭空造概念——在**已有资产里找一对天然的对立或互补**，再用一个广为人知的文化符号升华。这样的母题更真实、更易传播，也不容易显得"刻意"。

## 来源

把两个已经沉淀的 Skill 配成一对：

- `aigc-postmortem` —— 直译"尸检"，创作**之后**给作品做冷静解剖（先冻结事实，不许自我美化）。
- `maieutic-skill` —— maieutic 本义是苏格拉底的"接生术 / 助产"，创作**之前**帮你把模糊念头接生成清楚想法。

一头"助产"、一头"尸检"，正好是创作的**生与死**；再用《流浪地球2》图恒宇那句"我要给她完整的一生"串联。比原来干讲"什么是 Skill"有情感、有记忆点得多——这是一支技术科普片从"讲解"升格为"有温度"的关键。

## 验证状态

⚠️ 首次发现（单次，待二次验证）。

## 操作规则

1. 盘点已有资产（Skill / 作品 / 工具 / 概念），找一对**天然对立或互补**的（生／死、始／终、攻／守、轻／重、接生／验尸）。
2. 给这对配一个**反差词标签**（助产士 / 法医），反差越具体越抓人。
3. 用一个**广为人知的文化符号**（电影台词、典故）做情感升华。
4. 噱头"开头种、结尾收"，只点一两次——反复念就会变回"刻意"。

## 边界 / 反例

- 配对必须**真实存在**（两个 Skill 是真做出来的）。硬凑的配对会空洞。
- 文化符号要贴合主题；蹭热点式硬套会出戏。
- 升华句过量 = 煽情 = 刻意，与 [[知识科普片节奏诊断与重构_v1]] 的反"刻意"原则一致：克制是关键。
