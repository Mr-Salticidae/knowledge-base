## 一句话总结

跟"二人组"这类挑战 / 梗图,赢点在**选人 + 格式**,不在某一种画风:选一对**身材反差强、一眼能认、自带梗**的二人组,用一个**有张力的媒介风格**郑重画出来,落差即会心一笑;再在背景藏一个**第二眼母题**,顺手拿到震撼。**画风可换,这套锚点不变。**

---

## 选人三问(照着筛)

```
Q1 身材对得上吗? —— 体型差能不能直接长成题面那个二人组?
Q2 一眼能认吗?   —— 国民级文化预载,前 3 秒可识别?(见 入场票框架)
Q3 有梗吗?       —— 报出名字时,会不会"噗"一下 / 脊背一凉?
三个都中 → 就是它。
```

## 核心三组件(画风无关)

1. **身材反差 = 弱依赖脸的辨识钩子**:体型差(矮胖 ↔ 瘦高)+ 每人 1 件标志道具或配色。剪影时靠它免画脸,全彩时它仍是第一眼信号。
2. **有张力的媒介 + 反差**:用一个郑重、有张力的风格(史诗水墨 / JOJO 荒木 / ……)画接地气或搞笑的组合,落差 = 会心一笑;主体若自带悲壮或热血,张力叠加。
3. **第二眼母题**:背景藏一个与人物强相关、半融于背景的母题,制造二次发现(震撼来源)。

## 两种已验证执行

| | Ep.1 武大郎武松 | Ep.2 马里奥路易吉 |
|---|---|---|
| 画风 | 史诗水墨逆光剪影 | JOJO 荒木全彩漫画 |

| 第二眼 | 墨云隐虎 | 绿水管 + 飞散金币 |
| 调性 | 悲壮 → 震撼 | 热血 → 张力 |
| 工具 | GPT Image 2 | MJ niji 7 + 自有 personalize |

## 配套子法(Ep.2 新增)

### IP 绕障:风格点名,角色不点名
MJ 对商标角色(如马里奥/路易吉)有**输出形象拦截**("prompt 安全但生成图违规")。绕法:**画风可点名**(Hirohiko Araki / JoJo 是风格引用,放行),**角色用原型描述**(配色 + 体型 + 胡子,不写商标名)。辨识由配色体型扛,不靠名字。

### personalize 是质感临门一脚
同一条 prompt,叠不叠账号 personalize,是"能看"与"非常喜欢"的差别。personalize = 给账号训练的专属审美底盘,出图质感 / 张力的最后一脚。见 [[personalize与moodboard分工]]。

## 适用 / 不适用

| 适用 | 不适用 |
|---|---|
| 任意画风的二人组挑战 / 梗图(剪影或全彩皆可) | 需要精确面部表演 / 面部辨识的题 |
| 国民级、自带梗的组合(虚构 / 历史皆可) | 冷门、需大段解释才懂的组合 |
| 想同时要"会心一笑 + 震撼 / 张力" | 无反差、无第二眼可藏的题 |

- **二次验证 ⭐⭐**:Ep.1 水墨剪影 + Ep.2 JOJO 全彩,两种画风同套打法成立 → 核心锚点(选人三标准 + 身材反差 + 第二眼)与画风解耦,已确认。
- 升铁律 ⭐⭐⭐ 条件:再换一个题材 / 画风(如 黑白无常 浮世绘)第三次成立。
