# 高辨识二人组的剪影梗图法 v1

> 入档:2026-06-20
> 主线作品来源:`03_prompt模板库/02_案例复盘/2026-06-20_武大郎武松_史诗水墨二人组挑战复盘.md`
> 性质:**初次发现 ⚠️** —— 单案例(武大郎 × 武松 史诗水墨挑战图)提炼,未二次验证

---

## 一句话总结

跟"二人组剪影"这类挑战 / 梗图,**赢点在选人,不在画功**:选一对**身材反差强、一眼能认、自带梗**的二人组,用**史诗化媒介**郑重画出来,落差就是会心一笑;再在背景藏一个**第二眼母题**,顺手把震撼也拿到。

---

## 三个组件

### 1. 身材反差 = 免画脸的辨识钩子

剪影没有五官,辨识全靠**体型差 + 标志道具**。

- 体型轴:矮胖 ↔ 瘦高 / 高大 ↔ 娇小,是剪影里最强的一眼信号。
- 每人配 **1 件标志道具**作为身份锚:武大郎 = 炊饼扁担,武松 = 哨棒(+ 酒葫芦)。
- 道具要选"提名即认出"的国民符号,不要选需要解释的冷门物件。

> 用剪影做梗图时,先问:**不画脸,光看轮廓和手里的东西,能认出是谁吗?** 不能就换人或换道具。

### 2. 史诗化媒介反差(梗图的放大器)

把**接地气 / 搞笑**的二人组,用**史诗水墨 / 电影感 / 纪念碑构图**郑重其事地画出来。

- 越郑重,落差越好笑(给"卖炊饼的"上史诗逆光 = 笑点)。
- **加成项**:若组合本身有悲壮 / 深情底色(武松为兄复仇),震撼与笑点**叠加而非互斥**——这是最高级的版本,让人先笑后静。
- 手段:低角度仰拍、逆光纯剪影、泼墨光爆、大留白、背影朝向风暴(见 [[主体不看镜头律]])。

### 3. 第二眼隐喻(震撼的真正来源)

在背景里藏一个**"似散非散"的母题**,制造二次发现。

- 武大郎武松图:背景墨云里隐现一头**虎**(景阳冈打虎 / 命运)——一眼是兄弟,再看是猛虎。
- 母题要与人物**强相关**(虎之于武松),且**半融于背景**,不能抢主体。
- 这是低成本"回看钩子":第一眼会心一笑,第二眼脊背一凉。

---

## 适用 / 不适用

| 适用 | 不适用 |
|---|---|
| 剪影类 / 体型反差类二人组挑战、梗图 | 需要精确表情、面部辨识的题 |
| 国民级、自带梗的组合(虚构 / 历史皆可) | 冷门、需大段解释才懂的组合(辨识门槛太高) |
| 想同时要"会心一笑 + 震撼" | 纯写实、无反差的题 |

## 选人三问(照着筛)

```
Q1 身材对得上吗? —— 体型差能不能直接长成题面那个剪影?
Q2 一眼能认吗?   —— 国民级文化预载,前 3 秒可识别?(见 入场票框架)
Q3 有梗吗?       —— 报出名字时,会不会"噗"一下 / 脊背一凉?
三个都中 → 就是它。
```
