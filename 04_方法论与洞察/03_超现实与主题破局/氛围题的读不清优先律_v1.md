# 氛围题的"读不清"优先律 v1

> 入档：2026-06-22 · 来源：[[2026-06-22_不受控制的床_床长腿逃跑获奖图复盘]]
> 验证状态：⚠️ **首次发现**（单场 + 含创作者判断成分，v2 未投票）——仅作待观察假设，**不得指导下一次创作决策**。

---

## 核心

在**纪实超现实 / 氛围路线**里，画面的"读不太清"（动态模糊 / 半遮 / 夜色 / 抓拍感）**不是缺陷，是氛围资产**。它制造"我好像瞥见了什么、可又说不清"的真实感与不安感。

反过来：为"让人秒懂"而**过度交代超现实主体**（明确步态、锐化主体、完整暴露细节），会把"偶然抓拍到的真实瞬间"降级成"插画演示 / 摆拍说明图"，可信度和情绪反而垮掉。

一句话：**氛围题里，瞥见感 > 说明感。**

---

## 来源

「不受控制的床」获奖图（方向 ③ 床长腿夜里逃跑）：
- **v1**：床腿在动态模糊里、被夜色压暗、被拖曳的被单半遮——"读不太清" → **获奖（第一名）**。
- **v2**：刻意强化腿部清晰度（明确奔马步态、腿离地完整暴露、锐化腿部、模糊只留被单、被单不遮后腿）→ 创作者判断**不如 v1**。

> ⚠️ 注意边界：v2 **从未提交投票**。本律是「v1 实际获奖」+「创作者审美判断 v1>v2」两条证据合并，不是头对头票选。故只能 ⚠️ 首次发现，待二次验证。

## 操作规则

1. **先分类再决定要不要"修清楚"**：出图后若第一反应是"不够清楚"，先分辨——
   - "**题眼没被读出来**"（观众根本不知道这是什么）→ 要改，加清晰度。
   - "**氛围读不太清**"（一眼能感到不对劲、但细节糊）→ 可能是资产，**别动**。
2. **氛围题保留"读不清"的载体**：动态模糊、低照度夜色、浅景深、前景/拖曳物半遮主体、抓拍构图——这些是瞥见感的来源，不要为清晰度全删。
3. **留一个"那是什么？"的钩子**：氛围题的目标不是"秒懂全部信息"，而是"秒被氛围击中 + 留一层第二眼才解开的悬念"。
4. **把模糊放在该模糊的地方**：可以让主体一部分清晰（建立可信）、一部分糊（建立氛围），而不是整体锐化或整体糊。

---

## 反例 / 边界

- **不适用于视觉欺骗 / 反讽题**：松果手榴弹、垃圾高尔夫球这类靠"看错→反讽"取胜的题，**必须秒懂题眼**，清晰是友军；此时"读不清"= 失败。
- 与闪电战 [[闪电战的动作正在发生原则]]、"秒懂"诉求**不矛盾，而是划边界**：秒懂指"秒被击中"，不等于"秒看清所有细节"。
- 仅在**氛围 / 纪实超现实题**成立；题型判断错会两头踩空。
